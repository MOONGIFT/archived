GitHub Code SearchはGitHubのプロジェクトページにソースコード検索機能を提供するユーザスクリプトです。

  

GitHubをプロジェクトのコードリポジトリに利用しているプログラマーは多いと思いますが、さらにGitHubを便利にしてくれるUser Scriptを紹介します。GitHub Code Searchです。

  

[![](https://images.moongift.jp/2012/12/Screenshot-2012-12-10-12.47_thumb.png)](https://images.moongift.jp/2012/12/Screenshot-2012-12-10-12.47.png)  
**コード検索が追加されます！**

  

[![](https://images.moongift.jp/2012/12/Screenshot-2012-12-10-12.47.33_thumb.png)](https://images.moongift.jp/2012/12/Screenshot-2012-12-10-12.47.33.png)  
**検索結果はその場で表示されます。**

  

[![](https://images.moongift.jp/2012/12/Screenshot-2012-12-10-12.47.40_thumb.png)](https://images.moongift.jp/2012/12/Screenshot-2012-12-10-12.47.40.png)  
**言語を選んだりすると本体の検索結果ページへ。**

  

GitHub Code Searchをインストールしておけば該当リポジトリからだけ検索ができます。さらに右側には言語が出るのでJavaScriptやRubyなどといった言語で絞り込めばさらに良い結果が得られるでしょう。

  

GitHub Code SearchはJavaScript製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

  
  
  

**MOONGIFTはこう見る**

  

GitHubはソースコードのリポジトリであり、ソーシャルを概念に持ち込んだサービスですが、現在ではファイルの更新や新規作成に対応するなどIDE的な存在になりつつあります。さらにPushしたタイミングで別なサービスに反映させることもできるので統合的な開発環境とさえ言えます。

  

GitHub、Heroku、MongoHQなど開発からデプロイまで一通りの流れができてしまえばプログラマーやサービス提供までの流れが全て集約してしまいます。この一連の流れに加われるようなサービスを考え、開発してみると面白いのではないでしょうか。

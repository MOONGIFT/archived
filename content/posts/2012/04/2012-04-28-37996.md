CSS Arrow PleaseはCSSだけで作成する吹き出しデザインをオンライン上で行えるサービスです。

  

Twitterをはじめコミュニケーションが基本のサービスで欲しくなるのが吹き出し風のデザインです。そんなデザインをオンラインで簡単に作成できるのがCSS Arrow Pleaseです。

  

[![](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.31_thumb.png)](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.31.png)  
**トップページです。左に描かれている吹き出しデザインが右側のCSSで作成されています。**

  

[![](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.37_thumb.png)](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.37.png)  
**吹き出しの位置は上下左右から選べます。**

  

[![](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.44_thumb.png)](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.44.png)  
**色はカラーピッカーを使って自由に選択できます。**

  

[![](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.55_thumb.png)](https://images.moongift.jp/2012/04/Screenshot-2012-04-17-14.36.55.png)  
**カスタマイズしたパターンです。**

  

CSS Arrow Pleaseでは吹き出しの配置や色、線の太さなどを自由にカスタマイズできます。後は出来上がったスタイルをコピーして自分のサイトに組み込むだけです。

  

CSS Arrow PleaseはHTML/CSS製のオープンソース・ソフトウェア（MIT License）です。

  
  
  

**MOONGIFTはこう見る**

  

最近、この手のデザインモックアップをオープンソース化する試みが増えてきました。昔であれば関数やクラスをオープンソース化するようなイメージです。それがよりビジュアル的になっています。JavaScriptやCSS3によってデザインとプログラミングの垣根がなくなってきているのも要因として考えられます。

  

つまりオープンソースが従来のようにプログラマーだけのものではなくなってきているということです。Webデザイナーであっても面白いツールさえ作ればオープンソースの世界で名が売れる可能性があります。むしろ求められているのはデザインにあるとさえ言えるかもしれません。

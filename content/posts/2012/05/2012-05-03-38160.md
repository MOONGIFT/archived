jQuery Mobile Icon PackはjQuery Mobileで使えるアイコン集です。

  

jQuery Mobileを使ってスマートフォン最適化サイトを構築する際に使ってみたいのがjQuery Mobile Icon Packです。スマートフォンにマッチしたシンプルなアイコン集となっています。

**シンプルで使い勝手が良さそうです。**

  

アイコン画像は全て一つの画像となっており、CSS Spriteで切り出すようになっています。このアイコン群を使う限り読み込みは一度なので高速に表示が可能です。

  

jQuery Mobile Icon PackはCSS製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

  
  
  

**MOONGIFTはこう見る**

  

3Gを使ってどこでもインターネットが使えるとは言っても、速度はとても遅いです。スマートフォン向けWebサイトではHTML5を使って華やかにすることもできますが、画像を多用すると途端に遅くなります。極力絞り込んで使うのが良いでしょう。

  

見栄えがいいのは一見すると良いことですが、スマートフォンにおいてはお勧めできない場面もあります。一般的にスマートフォンでは情報をいち早く得るのが目的で、見栄えは二の次になりやすいです。そうした特性も理解した上でサイトを構築すべきです。

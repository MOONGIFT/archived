Web Reverse ReverseはReverse ReverseというiOS向けゲームの一部機能を再現したWebブラウザゲームです。

  

iOSで最も人気のあるジャンルと言えばゲームです。App Storeでも多数のゲームが人気を集めています。しかしそんな中だからこそ、Webブラウザで遊べるゲームを紹介します。iOS向けゲームのReverse ReverseをWebブラウザへ移植したWeb Reverse Reverseです。

  

[![](https://images.moongift.jp/2012/08/IMG_0355_thumb.png)](https://images.moongift.jp/2012/08/IMG_0355.png)  
**ゲーム開始画面です。Mobile Safariでのみ動作します。**

  

[![](https://images.moongift.jp/2012/08/IMG_0358_thumb.png)](https://images.moongift.jp/2012/08/IMG_0358.png)  
**ゲーム画面です。オセロの要領で画面上の白の数を指定された数にすればクリアです。この場合はゼロにすればOKです。**

  

[![](https://images.moongift.jp/2012/08/IMG_0356_thumb.png)](https://images.moongift.jp/2012/08/IMG_0356.png)  
**完了すると次のステージになります。**

  

[![](https://images.moongift.jp/2012/08/IMG_0357_thumb.png)](https://images.moongift.jp/2012/08/IMG_0357.png)  
**ゲームオーバーです。それまでのタッチ回数や時間、レベルで得点が決まるようです。**

  

Web Reverse Reverseは本家に比べるとエフェクトやゲームバランスが簡素化されていたり、一部のゲームモードのみが搭載されています。完全再現という訳ではありませんが、十分遊べるレベルのゲームになっています。ダウンロードせずにすぐに遊べるという点においては特にお勧めです。

  

Web Reverse ReverseはJavaScript製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

  
  
  

**MOONGIFTはこう見る**

  

スマートフォンのWebブラウザだけでも十分遊べるレベルのゲームが出来つつあります。オフラインキャッシュもありますが、それでもセーブなどを考えるとネットワークが必須ですが、それでも簡易的なゲームであれば十分楽しめます。また今後Flashが衰退することを考えるとHTML5への移行は必然と言えるでしょう。

  

iOS、Android向けのネイティブアプリと、HTML5アプリの二方向への進化が今後の必須と言えるでしょう。どちらが成功するかは分かりませんが、ゲーム自体を開発するよりもプラットフォーマーになる方が成功する可能性は高くなります。Apple、Google、Facebook、Zyngaなどが目指していますが、その一端を担うことだってできるはずです。

Bootstrap Image GalleryはTwitter社の開発したBootstrapを使ったイメージギャラリーソフトウェアです。

  

最近のWebサービスではTwitter社の開発した[Bootstrap](http://www.moongift.jp/2011/09/20110901-2/)をベースに提供するものが増えてきました。Bootstrap Image Galleryはそんな中、Bootstrapを使って開発されたイメージギャラリーソフトウェアとなっています。

  

[![](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.44.12_thumb.png)](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.44.12.png)  
**サムネイル一覧です。奇麗に並びます。**

  

[![](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.44.21_thumb.png)](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.44.21.png)  
**サムネイルをクリックしました。ローディングアイコンが表示されます。**

  

[![](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.44.26_thumb.png)](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.44.26.png)  
**拡大表示されました。**

  

[![](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.45.43_thumb.png)](https://images.moongift.jp/2012/01/Screenshot-2012-01-13-15.45.43.png)  
**フルスクリーンモードにも対応しています。**

  

Bootstrap Image GalleryはBootstrapのテンプレートに用いることで、ユーザビリティ高いインタフェースを容易に実現しています。またBootsrapのモーダルダイアログを拡張もしています。そうした点も参考になりそうです。

  
<!--more-->  

Bootstrap Image GalleryはJavaScript製、MIT Licenseのオープンソース・ソフトウェアです。

  
  
  

**MOONGIFTはこう見る**

  

かつては技術さえあれば売り方はどうにでもなると言われてきました。が、iPhoneで代表されるように技術力はもちろんのこと、そのデザインやマーケティング手法が正否を大きく分けるようになっています。特に見た目のインパクトにおいてデザインが担う役割は相当大きいと言えます。

  

そのため、Webデザインがうまくいかずに作ったWebサービス開発を頓挫させてしまうケースも少なくありません。そんな中、Bootstrapの効用は相当に大きいと言えます。さらにこうやってBootstrapを拡張していくライブラリやソフトウェアが増えれば、Webデザインにおけるベースのように扱われていくのではないでしょうか。

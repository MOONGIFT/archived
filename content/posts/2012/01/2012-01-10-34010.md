GitHub WatchersはGitHubのフォロワー数をバッジ表示できるソフトウェアです。

  

GitHubはプログラマーのソーシャルコミュニティです。ウォッチを通じてフォローできます。TwitterやFacebookでもこのフォロワーという存在が大きな意味を持ちます。そこで使ってみたいのがGitHub Watchersです。

  

[![](https://images.moongift.jp/2011/12/ScreenShot2011-12-29-14.51.24_thumb.png)](https://images.moongift.jp/2011/12/b26adbd63245d6fd08bfa081b9afa14c.png)  
**GitHubユーザのフォロワー数がボタン表示できます。**

  

[![](https://images.moongift.jp/2011/12/ScreenShot2011-12-29-14.51.30_thumb.png)](https://images.moongift.jp/2011/12/d45871249806b2298b425c8e6319d3a7.png)  
**実際にはリンクだけです。手軽に実装できそうです。**

  

[![](https://images.moongift.jp/2011/12/ScreenShot2011-12-29-14.51.36_thumb.png)](https://images.moongift.jp/2011/12/6e8b34e8950bc001c7bcc60984379805.png)  
**ロゴなし阪も用意されています。**

  

GitHub Watchersを自分のWebサイトに貼付けておけば、プログラミングコードに興味を持ってくれた人がフォローしてくれるきっかけになることでしょう。GitHub Watchersではロゴのあるなしやスタイル、テキストがカスタマイズできるようになっています。プログラマーの方はぜひ！

  
<!--more-->  

GitHub WatchersはJavaScript製のオープンソース・ソフトウェア（GPLまたはMIT License）です。

  
  
  

**MOONGIFTはこう見る**

  

オンラインアイデンティティを確立しようと考えた場合、このようなフォロワー数や友達数というのは大きな意味を持ってきます。フォロワーが多いとそれが呼び水になってさらにフォロワーを呼ぶきっかけになりますし、その数字の大小でオンラインにおける影響度を測れるようになります。

  

ある特定のコミュニティにおける影響度を測る上で、このフォロー/フォロワーという制度はとても有効です。自己顕示欲にも繋がりますので、バッジやアイコンを使ってアピールできるライブラリは人気が出やすいのではないでしょうか。そこからデータを集計できれば、面白い情報が集められそうです。

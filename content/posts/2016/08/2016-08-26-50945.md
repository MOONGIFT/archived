明るい画面を見続けていると目が疲れてしまいます。そこでプログラマなどはコード編集画面を暗くし、長時間画面を見ていても疲労しないようにしています。それは何もコードを書くときだけに限りません。
WebブラウザでGitHubのページを見る機会が多い人はGitHub Darkを使ってみると良いでしょう。

## GitHub Darkの使い方

GitHub DarkはGitHubを黒ベースのデザインに変更してくれます。

![](http://images.moongift.jp/2016/08/open-source-github%20dark-4_thumb.6bc80aa51efa091e39d4dd9d6ed15cd2.png "http://images.moongift.jp/2016/08/open-source-github%20dark-4.6bc80aa51efa091e39d4dd9d6ed15cd2.png")

README部分。

![](http://images.moongift.jp/2016/08/open-source-github%20dark-3_thumb.ac840e40fb7f71fd0d327cd179fc5bb1.png "http://images.moongift.jp/2016/08/open-source-github%20dark-3.ac840e40fb7f71fd0d327cd179fc5bb1.png")

リンクは緑になっています。

![](http://images.moongift.jp/2016/08/open-source-github%20dark-2_thumb.cb27f06ab59a661848c66a9888e4225f.png "http://images.moongift.jp/2016/08/open-source-github%20dark-2.cb27f06ab59a661848c66a9888e4225f.png")

コードも見やすくなっています。

![](http://images.moongift.jp/2016/08/open-source-github%20dark-1_thumb.3b3d308ef1af9482b2df4210545f1707.png "http://images.moongift.jp/2016/08/open-source-github%20dark-1.3b3d308ef1af9482b2df4210545f1707.png")

GitHub DarkはGitHubを表示している時だけ使えるスタイルシートなので、Stylishと組み合わせて使うようになっています。Google Chrome、Firefox、Safariで利用が可能です。コードを書いている時は黒ベースなのに、ブラウザに切り替えた途端に色ベースでコントラストがきついと感じている方は入れてみましょう。

GitHub DarkはCSSのオープンソース・ソフトウェア（MIT License）です。

React Nativeをこれから学んでいこうとしている方は多いでしょう。そんな時にイチから学ぶのも良いですが、すでにあるサンプルをベースに学んでいくのも良いでしょう。特にネットワークを伴う系は学べるところが多いです。
今回紹介するのはBBCNews React Native、アンオフィシャルなReact Native製のBBC Newsアプリです。

## BBCNews React Nativeの使い方

BBCNews React Nativeのメイン画面です。リストでフィードが表示されています。

![](http://images.moongift.jp/2016/06/open-source-bbcnews%20react%20native-3_thumb.1b3fa23a2addd44f478c160ccfa4ebfe.png "http://images.moongift.jp/2016/06/open-source-bbcnews%20react%20native-3.1b3fa23a2addd44f478c160ccfa4ebfe.png")

記事の詳細です。動画の再生もできます。

![](http://images.moongift.jp/2016/06/open-source-bbcnews%20react%20native-1_thumb.acd84788dd4b6574f67cce24e8532c8b.png "http://images.moongift.jp/2016/06/open-source-bbcnews%20react%20native-1.acd84788dd4b6574f67cce24e8532c8b.png")

BBCNews React Nativeは特に特別な機能はなく、一覧と詳細画面くらいしかありません。それだけにアプリの構造を理解するのは簡単そうです。これをベースに機能を追加して、自分だけのメディアアプリを作ってみても良さそうです。

BBCNews React NativeはReact/JavaScript製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

[joeltrew/BBCNews-React-Native: An unofficial basic implementation of the BBC News app built using React Native](https://github.com/joeltrew/BBCNews-React-Native)

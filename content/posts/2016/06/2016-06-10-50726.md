Bootstrapが最もよく使われるのは個人のWebサービス、そして企業の管理画面ではないでしょうか。そこまでデザインがよくある必要はないですが、見逃し防止の一覧性やグラフの見やすさなどが求められます。
そんなBootstrapを使った管理画面テンプレート、Bootstrap Clean Dashboard Themeを紹介します。情報が一カ所に集中したダッシュボードが作れそうです。

## Bootstrap Clean Dashboard Themeの使い方

デモ画面です。殆どのボタンは機能しません。

![](http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-6_thumb.8ae9fbce0e99b27e1667f66c04786900.png "http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-6.8ae9fbce0e99b27e1667f66c04786900.png")

グラフもあります。各コンテンツはボックスに入っています。

![](http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-5_thumb.7ad0565142f315572cb851bd302e18a0.png "http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-5.7ad0565142f315572cb851bd302e18a0.png")

ボックスをクリックするまでは折りたたんでおくといった表示もできます。

![](http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-4_thumb.bf4ade8a26f0d026f0c73b62cedb0dd4.png "http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-4.bf4ade8a26f0d026f0c73b62cedb0dd4.png")

フォーム。

![](http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-3_thumb.971066e32ff3e8bd55328d5eda30656c.png "http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-3.971066e32ff3e8bd55328d5eda30656c.png")

アドレス帳風。

![](http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-2_thumb.d38ac88349fcf43bd37b04f76fa3b69b.png "http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-2.d38ac88349fcf43bd37b04f76fa3b69b.png")

ログイン。

![](http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-1_thumb.1a76003e7d00d84112d16aec41be4003.png "http://images.moongift.jp/2016/05/open-source-bootstrap%20clean%20dashboard%20theme-1.1a76003e7d00d84112d16aec41be4003.png")

Bootstrap Clean Dashboard Themeはダッシュボードなので、管理画面上にたくさんの情報を見やすく配置するのが求められるでしょう。アイコンも組み合わせることで、情報が込み入ったりすることなく、整頓して表示できるはずです。

Bootstrap Clean Dashboard ThemeはBootstrap製のオープンソース・ソフトウェア（MIT License）です。

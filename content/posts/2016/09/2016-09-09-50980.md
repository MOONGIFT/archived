GitHubは開発者にとって個人の技術力であったり、オープンソースへの関わり方をアピールする大事なサービスとなっています。そうなると考えなければならないのが情報の見せ方ではないでしょうか。
そこで使ってみたいのがGitHub Explorerです。GitHub標準とは違う、格好良い表示を実現してくれます。

## GitHub Explorerの使い方

自分のページを表示してみました。レスポンシブデザインでスマートフォンにも対応しています。

![](http://images.moongift.jp/2016/08/open-source-github%20explorer-4_thumb.8b850782c8fe0d49db20e2dff5e62f72.png "http://images.moongift.jp/2016/08/open-source-github%20explorer-4.8b850782c8fe0d49db20e2dff5e62f72.png")

検索でユーザを探せます。

![](http://images.moongift.jp/2016/08/open-source-github%20explorer-3_thumb.9c26f5a5d7ded0ed844a9ee4f5214357.png "http://images.moongift.jp/2016/08/open-source-github%20explorer-3.9c26f5a5d7ded0ed844a9ee4f5214357.png")

リポジトリの一覧。

![](http://images.moongift.jp/2016/08/open-source-github%20explorer-2_thumb.585d3623a3785712ccedd10c1031c870.png "http://images.moongift.jp/2016/08/open-source-github%20explorer-2.585d3623a3785712ccedd10c1031c870.png")

リポジトリの検索もできます。

![](http://images.moongift.jp/2016/08/open-source-github%20explorer-1_thumb.dae0125f93de441bd79e34737287086d.png "http://images.moongift.jp/2016/08/open-source-github%20explorer-1.dae0125f93de441bd79e34737287086d.png")

GitHub Explorerはデザインがすっきりとしており、格好良いです。Web APIを介して取得しているのでSEO的な効果は期待できません。あくまでも自分のサイトを見ている「人」向けに利用することになるでしょう。

GitHub Explorerはnode/JavaScript製のオープンソース・ソフトウェア（ISC License）です。

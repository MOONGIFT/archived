Webサイトの基本と言えばCMS（コンテンツ・マネージメント・システム）です。しかしすべて同じCMSで良いかと言えばそうではなく、C（コンテンツ）によってシステムの選定は分かれるべきです。
今回紹介するIdeaSpace、今話題性のあるVR向けコンテンツを管理するCMSです。

## IdeaSpaceの使い方

管理画面です。テーマがあり、一般的なVRオブジェクトの他、360度動画向けのテーマもあります。

![](http://images.moongift.jp/2016/06/open-source-ideaspace-5_thumb.07c1292e6827a07701ede51610ce6535.png "http://images.moongift.jp/2016/06/open-source-ideaspace-5.07c1292e6827a07701ede51610ce6535.png")

新規コンテンツ追加画面です。

![](http://images.moongift.jp/2016/06/open-source-ideaspace-4_thumb.dfbedb5fedf5b7b159153e9dfdeada43.png "http://images.moongift.jp/2016/06/open-source-ideaspace-4.dfbedb5fedf5b7b159153e9dfdeada43.png")

コンテンツの表示画面です。

![](http://images.moongift.jp/2016/06/open-source-ideaspace-3_thumb.30b1042954f493d5ff49aff4d84d524c.png "http://images.moongift.jp/2016/06/open-source-ideaspace-3.30b1042954f493d5ff49aff4d84d524c.png")

VRコンテンツに文字を載せています。

![](http://images.moongift.jp/2016/06/open-source-ideaspace-2_thumb.4e17d8e448c5c7ecf0438ba0657eddff.png "http://images.moongift.jp/2016/06/open-source-ideaspace-2.4e17d8e448c5c7ecf0438ba0657eddff.png")

こちらは360度動画です。

![](http://images.moongift.jp/2016/06/open-source-ideaspace-1_thumb.62f27d5530fd2603a87f2950d0b94102.png "http://images.moongift.jp/2016/06/open-source-ideaspace-1.62f27d5530fd2603a87f2950d0b94102.png")

IdeaSpaceはモバイル、デスクトップの両方に対応しています。さらにMozillaのA-FrameというVR用のフレームワークを使うことで効率的な開発ができるようになります。VR向けサイトを作る場合はベースにしてみても良さそうです。

IdeaSpaceはPHP製のオープンソース・ソフトウェア（GPL）です。

[IdeaSpace - Open source PHP content management system for the virtual reality web (WebVR)](https://www.ideaspacevr.org/)
[IdeaSpaceVR/IdeaSpace: PHP Content management system (CMS) for the virtual reality web (WebVR)](https://github.com/ideaspacevr/ideaspace)

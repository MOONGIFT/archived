GitHubでの活動はプログラマにとっての一つのステータスになってきています。知り合った人がGitHubでどれくらい開発を行っているか見たり、寄せられたIssueに対して発言している人が誰か見たりしたことはないでしょうか。
そんなGitHub上での人となりを確認するのに便利なのがGitHub Hovercardです。わざわざクリックせずにユーザやリポジトリの情報がチェックできます。

## GitHub Hovercardの使い方

GitHub HovercardはFirefox/Google Chromeのアドオン（機能拡張）として提供されています。

![](http://images.moongift.jp/2015/10/open-source-github-hovercard-3_thumb.9c1f3ab352b4ea52e31f8209a1a89bc2.png "http://images.moongift.jp/2015/10/open-source-github-hovercard-3.9c1f3ab352b4ea52e31f8209a1a89bc2.png")

インストールするとリポジトリへのマウスオーバーでフローティングが出るようになります。

![](http://images.moongift.jp/2015/10/open-source-github-hovercard-2_thumb.28ae569414536cc79f81396a4cd3da62.png "http://images.moongift.jp/2015/10/open-source-github-hovercard-2.28ae569414536cc79f81396a4cd3da62.png")

ユーザアイコンやユーザ名にマウスオーバーしても情報が出ます。

![](http://images.moongift.jp/2015/10/open-source-github-hovercard-1_thumb.d7917a2ff35a21275558e70c31a2ca18.png "http://images.moongift.jp/2015/10/open-source-github-hovercard-1.d7917a2ff35a21275558e70c31a2ca18.png")

GitHub Hovercardを使うとGitHub内の任意のリンクに対して情報が追加されます。リンクをクリックしないで良いのはとても楽です。Facebookでもユーザ名のところでフローティングが出ますが、ああいった感じの使いやすさになるでしょう。

GitHub HovercardはGoogle Chrome/Firefox用のオープンソース・ソフトウェア（MIT License）です。

グルーポンが自社のサービスにGnomeとつけた上、商標を取ろうとして炎上したり、AppleがSwiftと言う言語（既に同名のプログラミング言語があるのに！）を作ったりとネーミングについては万一重複していたりすると後々やっかいな問題につながります。
そこでもしあなたが新しいプロジェクトをはじめようと思ったら使って欲しいのがOpen source project name checkerです。

## Open source project name checkerの使い方

![このように有名な各プロジェクトホスティングサービスを使ってネーミングチェックを行ってくれます。](http://images.moongift.jp/2015/02/Screenshot%202015-02-15%2017.33.06_thumb.109995b64b3881348cfcd7e40e7581db.png "http://images.moongift.jp/2015/02/Screenshot%202015-02-15%2017.33.06.109995b64b3881348cfcd7e40e7581db.png")

![全部笑い顔だったら重なっていないという判断です。](http://images.moongift.jp/2015/02/Screenshot%202015-02-15%2017.33.49_thumb.5e2562b4de4c0fe9d8306449d4e26d80.png "http://images.moongift.jp/2015/02/Screenshot%202015-02-15%2017.33.49.5e2562b4de4c0fe9d8306449d4e26d80.png")

もちろんユニークなのがベストで、かつドメインも空いていたりすれば良いプロジェクト名になるでしょう。せっかく格好良いネーミングを考えるならば、既存のプロジェクト名と重なっていないかどうかのチェックは必須でしょう。

Open source project name checkerはnode/JavaScript製のオープンソース・ソフトウェア（MIT License）です。

Reactへの注目がどんどん高まっています。Webのビューライブラリという枠を越えて、React Nativeによってスマートフォンアプリ開発にも使えるようになっています。特にコンセプトであるLearn once, run everywhere（一度覚えればどこでも動く）が素晴らしい考えと言えます。
そんなReactを次のターゲットとしてデスクトップでも動くようにしたのがReact Desktopです。Mac OSX/Windowsに対応しています。

## React Desktopの使い方

React Desktopのデモです。Webでも表示できるのがすごいです。

![](http://images.moongift.jp/2015/10/open-source-react-desktop-5_thumb.4d31577c20adb7485ae5856ba4772ca0.png "http://images.moongift.jp/2015/10/open-source-react-desktop-5.4d31577c20adb7485ae5856ba4772ca0.png")

コントロールもちゃんと動きます。

![](http://images.moongift.jp/2015/10/open-source-react-desktop-4_thumb.33add1e36542d9e13847fcd346b6824e.png "http://images.moongift.jp/2015/10/open-source-react-desktop-4.33add1e36542d9e13847fcd346b6824e.png")

そしてWindows 10でも使えます。

![](http://images.moongift.jp/2015/10/open-source-react-desktop-3_thumb.9b792372cc4e034f680e87748e4d2bf5.png "http://images.moongift.jp/2015/10/open-source-react-desktop-3.9b792372cc4e034f680e87748e4d2bf5.png")

フォームもまさにWindows 10といった感じです。

![](http://images.moongift.jp/2015/10/open-source-react-desktop-2_thumb.931595261f10f20176a80cf498db587b.png "http://images.moongift.jp/2015/10/open-source-react-desktop-2.931595261f10f20176a80cf498db587b.png")

メニューを隠すこともできます。

![](http://images.moongift.jp/2015/10/open-source-react-desktop-1_thumb.3c88b01db06bd5869fca894e5cad47d2.png "http://images.moongift.jp/2015/10/open-source-react-desktop-1.3c88b01db06bd5869fca894e5cad47d2.png")

ElectronはHTML5を使ってデスクトップアプリを作ることができます。それはそれで開発が楽になるのですが、React Desktopのようにネイティブのコンポーネントに対応することで速度やJavaScriptだと苦手な巨大なファイルを扱うような処理もこなせるようになりそうです。

なお、React DesktopはMac OSX El CapitanおよびWindows 10対応とのことで、最新の環境向けに作るならと割り切る必要がありそうです。

React DesktopはMac OSX/Windows用のオープンソース・ソフトウェア（MIT License）です。

React Nativeは面白い仕組みですが、すべてのアプリ開発に適用できるわけではないでしょう。しかしシンプルなアプリであれば利用できる可能性がとても高いです。そのためにどんなアプリに向いているか、試す必要があるでしょう。
今回はその一例として[HackerNews-React-Native](https://github.com/iSimar/HackerNews-React-Native)を紹介します。HackerNewsをReact Nativeを使ってiOSアプリ化しています。

## HackerNews-React-Nativeの使い方

アプリを使う際には npm start しておく必要があります。まずは記事一覧が表示されます。

![](http://images.moongift.jp/2015/04/open-source-hackernews-react-native-4_thumb.70bdcfc45d57f61ca4d61d70cba10345.png "http://images.moongift.jp/2015/04/open-source-hackernews-react-native-4.70bdcfc45d57f61ca4d61d70cba10345.png")

一覧をクリックすると詳細表示にスライドします。

![](http://images.moongift.jp/2015/04/open-source-hackernews-react-native-3_thumb.e4421a38aceee388810902fd47a775e3.png "http://images.moongift.jp/2015/04/open-source-hackernews-react-native-3.e4421a38aceee388810902fd47a775e3.png")

コメントはタップすると読み込まれます。

![](http://images.moongift.jp/2015/04/open-source-hackernews-react-native-2_thumb.12f72f5b7fea52faeffa9991cae48847.png "http://images.moongift.jp/2015/04/open-source-hackernews-react-native-2.12f72f5b7fea52faeffa9991cae48847.png")

さらに実際のページを見ることもできます。

![](http://images.moongift.jp/2015/04/open-source-hackernews-react-native-1_thumb.a0c6622d55a45d21a877d3ac69a37527.png "http://images.moongift.jp/2015/04/open-source-hackernews-react-native-1.a0c6622d55a45d21a877d3ac69a37527.png")

HackerNews-React-Nativeはシンプルな作りなので、まさにReact Native向きと言えるかも知れません。とはいえ作りはしっかりとコードが分かれて作られていますので参考になる点が多いかと思います。ぜひ参考にしてください。

HackerNews-React-NativeはJavaScript/React Native製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

ブログなどで画像を一覧で表示したり、記事中に写真を埋め込んでいるのはよく見る光景です。そんな時、画像をクリックするとフローティングで拡大表示されます。また、次の写真に移動することもできるでしょう。
そんな見せ方はCSSとJavaScriptを使ってこそできるものだと思っていましたが、CSS Galleryは違います。スタイルシートだけで実現してしまうのです。

## CSS Galleryの使い方

CSS Galleryのデモです。写真の並び方も格好良いです。

![](http://images.moongift.jp/2015/11/open-source-css-gallery-6_thumb.f67c7fdb898cb2460efae37080f49d6f.png "http://images.moongift.jp/2015/11/open-source-css-gallery-6.f67c7fdb898cb2460efae37080f49d6f.png")

クリックでモーダル表示。これもスタイルシートのみです。

![](http://images.moongift.jp/2015/11/open-source-css-gallery-5_thumb.440c7355189db59eed21d8dfb012563f.png "http://images.moongift.jp/2015/11/open-source-css-gallery-5.440c7355189db59eed21d8dfb012563f.png")

次の写真に切り替えもできます。

![](http://images.moongift.jp/2015/11/open-source-css-gallery-4_thumb.f97683705969d7037adbf817030ab754.png "http://images.moongift.jp/2015/11/open-source-css-gallery-4.f97683705969d7037adbf817030ab754.png")

写真のリスト表示もサポートされています。

![](http://images.moongift.jp/2015/11/open-source-css-gallery-3_thumb.fa8862752e43052f51b9bc116e924bbd.png "http://images.moongift.jp/2015/11/open-source-css-gallery-3.fa8862752e43052f51b9bc116e924bbd.png")

リスト表示でもモーダル表示できます。

![](http://images.moongift.jp/2015/11/open-source-css-gallery-2_thumb.8d1fb9186e9327c35af1273b52017879.png "http://images.moongift.jp/2015/11/open-source-css-gallery-2.8d1fb9186e9327c35af1273b52017879.png")

実際に操作しているところです。

![](http://images.moongift.jp/2015/11/open-source-css-gallery-1.b77d1e6b7775bc0e51f1627a0c153203.gif "http://images.moongift.jp/2015/11/open-source-css-gallery-1.b77d1e6b7775bc0e51f1627a0c153203.gif")

CSS Galleryが良いのはJavaScriptがオフになっていても使えるということです。JavaScriptを使えば力技で解決できますが、それをあえてスタイルシートだけで実現するところに面白さが感じられます。

CSS GalleryはCSS製のオープンソース・ソフトウェア（MIT License）です。

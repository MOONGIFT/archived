Visual Studioと言えば古くからある優秀なIDEです。若干重たいのが気になりますが、それでもVisual Studioの上だけで生活できているという人も多いことでしょう。Windows用のソフトウェアからWeb開発まで何でもこなせる多機能なIDEです。
そんなVisual Studioのブランドを背負って登場した新しいプログラミングエディタがVisual Studio Codeです。AtomをベースにしたWeb開発用エディタとなっています。

## Visual Studio Codeの使い方

Visual Studio Codeの画面です。

![](http://images.moongift.jp/2015/12/open-source-vscode-7_thumb.b321b2acaed2ad5b3883be2d8b90f16d.png "http://images.moongift.jp/2015/12/open-source-vscode-7.b321b2acaed2ad5b3883be2d8b90f16d.png")

フォルダを開くこともできます。

![](http://images.moongift.jp/2015/12/open-source-vscode-6_thumb.c27e1fb34a3bb901979717f8a6f1343a.png "http://images.moongift.jp/2015/12/open-source-vscode-6.c27e1fb34a3bb901979717f8a6f1343a.png")

Git機能がデフォルトで入っているのが便利です。

![](http://images.moongift.jp/2015/12/open-source-vscode-5_thumb.10b6e54bb14a4c652d8c4426e82a6c67.png "http://images.moongift.jp/2015/12/open-source-vscode-5.10b6e54bb14a4c652d8c4426e82a6c67.png")

差分もVisual Studio Code上で確認できます。

![](http://images.moongift.jp/2015/12/open-source-vscode-4_thumb.f6196f8829ac7f4fb7ed5c6e560a4bbe.png "http://images.moongift.jp/2015/12/open-source-vscode-4.f6196f8829ac7f4fb7ed5c6e560a4bbe.png")

シンタックスハイライトは多くの言語に対応しています。

![](http://images.moongift.jp/2015/12/open-source-vscode-3_thumb.56ad42325d22bd03832741bd1ffc17c7.png "http://images.moongift.jp/2015/12/open-source-vscode-3.56ad42325d22bd03832741bd1ffc17c7.png")

変数名の入力補完にも対応しています。

![](http://images.moongift.jp/2015/12/open-source-vscode-2_thumb.3ce6eb3072e91b602d48d1fcff412aa1.png "http://images.moongift.jp/2015/12/open-source-vscode-2.3ce6eb3072e91b602d48d1fcff412aa1.png")

日本語のコード検索も可能です。

![](http://images.moongift.jp/2015/12/open-source-vscode-1_thumb.907198ab735d8dc6ade2909f026c8c83.png "http://images.moongift.jp/2015/12/open-source-vscode-1.907198ab735d8dc6ade2909f026c8c83.png")

Visual Studio Codeはnode.jsのデバッグ機能も入っており、Webフロントエンドの開発からサーバサイドまでこなすことができます。Atomより高速という話もあります。別なエディタからAtomに乗り換えようと思っていた方はVisual Studio Codeも試してみてはいかがでしょう。

Visual Studio CodeはWindows/Mac OSX/Linux用のオープンソース・ソフトウェア（MIT License）です。

ソフトウェアは単機能になればなるほど専門性を増し、それがユーザの心に刺さりやすくなります。汎用的なソフトウェアは多くのことができそうですが、かえって本当に欲しい一つの機能がなかったりするものです。
今回紹介するのは[Mobirise Mobile Website Builder](http://mobirise.com/)、スマートフォンやタブレット向けのWebサイト専用の構築ツールになります。

## Mobirise Mobile Website Builderの使い方

Mobirise Mobile Website BuilderはWindows、Mac OSX版が提供されています。立ち上げるとデモのWebサイトが表示されます。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-9_thumb.e8bd824026c51c03a177343a6486a37a.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-9.e8bd824026c51c03a177343a6486a37a.png")

右下のボタンをクリックするとブロックが表示されます。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-8_thumb.bb796c9f8afab4ab5b96543034092dcf.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-8.bb796c9f8afab4ab5b96543034092dcf.png")

様々なパーツが予め用意されています。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-7_thumb.37ade16d9bc670e93404d5f631918528.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-7.37ade16d9bc670e93404d5f631918528.png")

スマートフォンの幅にしてプレビューができます。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-5_thumb.1d579cd2c6dc43e655e27bbca18f0dc4.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-5.1d579cd2c6dc43e655e27bbca18f0dc4.png")

ページの追加もできます。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-4_thumb.ab326e3a7a68760d05b9f40ec8d0ecd8.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-4.ab326e3a7a68760d05b9f40ec8d0ecd8.png")

パラメータを変更することで細かな調整も可能です。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-3_thumb.3cec141b7f11ba703e232a25770edbd3.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-3.3cec141b7f11ba703e232a25770edbd3.png")

文言はWYSIWYGに変更ができます。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-2_thumb.b298a0d8723ac0719273df7cf92eaf3c.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-2.b298a0d8723ac0719273df7cf92eaf3c.png")

できあがったコンテンツはFTPまたはGoogle Driveにアップロードができます。

![](http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-1_thumb.08fb4bfebc9df8160f7dc90c6075fc9e.png "http://images.moongift.jp/2015/07/open-source-mobirise-mobile-website-builder-1.08fb4bfebc9df8160f7dc90c6075fc9e.png")

Mobirise Mobile Website Builderは主にSPA（シングルページアプリケーション）用と思われますが、複数のページを作ることもできます。デザインは最初からできあがっていますので、後はパーツを組み合わせていくだけで簡単にスマートフォン、タブレットに対応したレスポンシブなWebサイトができあがるでしょう。

Mobirise Mobile Website BuilderはWindows/Mac OSX用のソフトウェアです。

[Mobirise Mobile Website Builder](http://mobirise.com/)

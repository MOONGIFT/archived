React Nativeに熱い視線が集まっています。Reactを使ってWebベースのようにiOSのネイティブアプリが開発できるのが魅力と言えます。制限も多いとは思いますが、それでもどんなことができるのかみんなが試行錯誤しながら試しているのが現状です。
そんなReact Nativeを使って作られたアプリが[Dribbble app built with React Native](https://github.com/catalinmiron/react-native-dribbble-app)です。ごくシンプルなDribbbleビューワーアプリとなっています。

## Dribbble app built with React Nativeの使い方

Dribbble app built with React NativeはReact Nativeらしい、ごくシンプルなUIのアプリとなっています。興味深いのはパララックスエフェクトくらいかも知れません。しかしそれが以外とDribbbleの雰囲気とあっているので、ビューワーアプリとして十分な作りとなっています。

Dribbble app built with React NativeはReact Native/JavaScript製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

AndroidはiPhoneと違ってデスクトップとつないでファイルを送り合うのも簡単です。開発時にAPKファイルを送ったり、写真や動画をコピーしたりします。しかしそのファイルをAndroid側で開く際にちょっと面倒だなと思うことがあります。
シンプルで使い勝手の良いファイルマネージャを探している方はFile Manager Betaを使ってみてはいかがでしょう。

## File Manager Betaの使い方

インストールは[Google Play](https://play.google.com/store/apps/details?id=com.michaldabski.filemanager)からできます。

File Manager Betaは余計な機能はなく、フォルダの作成とファイルの閲覧、お気に入り登録くらいの機能しかありません。それだけにシンプルで使い勝手が良いです。オープンソースなのでコードを見て学べる点も多そうです。

File Manager BetaはAndroid用、MIT Licenseのオープンソース・ソフトウェアです。

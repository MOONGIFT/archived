Bootstrap Application WizardはBootstrap用のアプリケーションウィザードフローを提供するライブラリです。

  
  

Bootstrapを使ってWebサービスを提供するサイトが多くなってきました。今回はその一つ、Bootstrapを使ってウィザード風の流れを実現してくれるライブラリ、Bootstrap Application Wizardを紹介します。

  

[![](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.20.17_thumb.1360764746.png)](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.20.17.1360764746.png)  
**フローティングでウィザードが表示されます。**

  

[![](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.20.38_thumb.1360764750.png)](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.20.38.1360764750.png)  
**次へを押して順番に進めていきます。**

  

[![](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.20.47_thumb.1360764755.png)](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.20.47.1360764755.png)  
**左下のステータスバーが徐々に進んでいます。**

  

[![](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.21.25_thumb.1360764759.png)](http://images.moongift.jp/2013/02/Screenshot 2013-02-13 22.21.25.1360764759.png)  
**完了しました！**

  

Bootstrap Application Wizardではwizardというメソッドを使って一つ一つをスライド風に切り替えながら処理を選べるようになります。入力チェックも内蔵されています。そして最後にデータが飛んでくるのでサーバ側で処理して返却すれば良いだけです。とても手軽です。

  

Bootstrap Application WizardはJavaScript製、MIT Licenseのオープンソース・ソフトウェアです。

  
  
  

**MOONGIFTはこう見る**

  

Bootstrapは登場以来人気が衰えることなく、むしろどんどん採用されるケースが広がっています。これまで小さなウィジェットでBootstrapを強化するソフトウェアは数多くありましたが、さらに本格的なものが出てきたかという感があります。

  

こうやって多彩なコンポーネントが次々に登場すれば、開発者はそれらの部品を組み合わせるだけで済むようになります。既にBootstrap対応のテーマ配布サイトも数多くありますので、それらを使えばあっという間に見栄えのいいサイトができあがります。この仕組みを使わない手はないでしょう。

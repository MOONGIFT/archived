jQuery Mobile Flat UI ThemeはjQuery Mobile用のフラットUIテーマです。

  
  

スマートフォン向けのWebサイトをjQuery Mobileで構築しているなら試したいのがjQuery Mobile Flat UI Themeです。最近流行のフラットなUIをサポートしたテーマです。

  

[![](http://images.moongift.jp/2013/04/Screenshot 2013-04-08 13.32.40_thumb.1365400555.png)](http://images.moongift.jp/2013/04/Screenshot 2013-04-08 13.32.40.1365400555.png)  
**ボタン類。アイコンも使えていい感じです。**

  

[![](http://images.moongift.jp/2013/04/Screenshot 2013-04-08 13.33.17_thumb.1365400559.png)](http://images.moongift.jp/2013/04/Screenshot 2013-04-08 13.33.17.1365400559.png)  
**リストと見出し付きコンテンツ。見出しをクリックすると内容が表示されます。**

  

[![](http://images.moongift.jp/2013/04/Screenshot 2013-04-08 13.33.30_thumb.1365400563.png)](http://images.moongift.jp/2013/04/Screenshot 2013-04-08 13.33.30.1365400563.png)  
**アイコンやツールバー。スライダーもあります。**

  

主なカラーは黄色、緑系2色、赤、青そして白黒となっています。色を使いすぎないシンプルさ、影などがないフラットな見た目が好感です。

  

jQuery Mobile Flat UI ThemeはjQuery Mobile用、MIT Licenseのオープンソース・ソフトウェアです。

  
  
  

**MOONGIFTはこう見る**

  

有名なフレームワークをその作法に則って使っておくと、デザインをがらっと変更したい場合に優位です。WordPressなども世界中にある多彩なテーマを使いこなせば、サイトのリニューアルも簡単にできるはずです。特にそれが時代に合わせて変更しているならば陳腐化する前に乗り換えないといけないでしょう。

  

デザインは特にその時々に合わせた流行があります。Web2.0という単語が飛び交った時にはシンプルなデザインが流行りましたが、徐々に角丸になり、淡いデザインが流行った後、現在のようなUIに人気が集まっています。乗り遅れないためにもクライアントサイドのフレームワークを活用するのが良いでしょう。

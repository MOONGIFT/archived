iOSではSafariが基本のブラウザですが、他にも幾つかサードパーティー製のブラウザが存在します。それぞれ特徴があり、高速でブラウジングできたり、タブの扱いが違ったりします。
今回はその一つ、オープンソースのBrave Web Browserを紹介します。

## Brave Web Browserの使い方

最初に表示されるチュートリアルです。

![](http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-6_thumb.79c656ef4e39b3ce87d0621927da2a2c.png "http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-6.79c656ef4e39b3ce87d0621927da2a2c.png")

アドブロック機能がデフォルトで有効です。

![](http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-5_thumb.a58fe994f3a73cfafc31f57ab577ac5a.png "http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-5.a58fe994f3a73cfafc31f57ab577ac5a.png")

メニューです。アドブロックの有効/無効の切り替えができます。

![](http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-3_thumb.604982e4ebc7db47140ca902d1ed7548.png "http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-3.604982e4ebc7db47140ca902d1ed7548.png")

切り替えると無効になります。

![](http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-4_thumb.5de5a816dcd0381d13a4ad26f27e583a.png "http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-4.5de5a816dcd0381d13a4ad26f27e583a.png")

タブ機能が見やすいです。

![](http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-1_thumb.26ce92d013a8e3021ae4552a561b8a94.png "http://images.moongift.jp/2017/03/open-source-brave%20web%20browser-1.26ce92d013a8e3021ae4552a561b8a94.png")

標準のSafariに比べて2〜4倍で表示できるとのことです（実際には多少速いかもといった感じですが）。ベースはFirefox for iOSとなっています。標準のSafariに不便さを感じている方は乗り換えてみてはいかがでしょう。

Brave Web BrowserはiOS用のオープンソース・ソフトウェア（Mozilla Public License）です。

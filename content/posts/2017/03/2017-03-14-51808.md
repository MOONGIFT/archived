ノートアプリは一つ用意しておくと便利です。気がついた時にささっとメモを取っておけば、後で振り返ることもできます。頭の中で覚えておこうとしても忘れてしまうものなので、すぐにメモするのが大事なのです。
今回紹介するオープンソース・ソフトウェアはQOwnNotes、マルチプラットフォームで動作するノートアプリです。

## QOwnNotesの使い方

最初に保存場所を指定します。

![](http://images.moongift.jp/2017/02/open-source-qownnotes-6_thumb.2db4b2dff94762e69364f1a605bbbf9a.png "http://images.moongift.jp/2017/02/open-source-qownnotes-6.2db4b2dff94762e69364f1a605bbbf9a.png")

メイン画面です。左側にノートの一覧、右側がエディタになります。

![](http://images.moongift.jp/2017/02/open-source-qownnotes-5_thumb.f8e5298aa32dd76a72a8cee0df6bf4b0.png "http://images.moongift.jp/2017/02/open-source-qownnotes-5.f8e5298aa32dd76a72a8cee0df6bf4b0.png")

新しいノートを書きます。

![](http://images.moongift.jp/2017/02/open-source-qownnotes-4_thumb.53c7aab6ad87b172448f2984f80457e5.png "http://images.moongift.jp/2017/02/open-source-qownnotes-4.53c7aab6ad87b172448f2984f80457e5.png")

フルバージョン。タグやプレビューなども表示されます。

![](http://images.moongift.jp/2017/02/open-source-qownnotes-3_thumb.6612e688106a4d0c3ef10a9666d48fc1.png "http://images.moongift.jp/2017/02/open-source-qownnotes-3.6612e688106a4d0c3ef10a9666d48fc1.png")

設定画面です。TodoはCalDAVサーバを使います。

![](http://images.moongift.jp/2017/02/open-source-qownnotes-2_thumb.f0dbe0ec897a3e53e17e8e96668a8532.png "http://images.moongift.jp/2017/02/open-source-qownnotes-2.f0dbe0ec897a3e53e17e8e96668a8532.png")

QOwnNotesはローカルだけでも動きますが、ownCloudと連携させるのが良いようです。そうすればデータをクラウドに保存したり、カレンダーのデータを使ったTodo管理もできるようになります。

QOwnNotesはWindows/macOS/Linux用のオープンソース・ソフトウェア（GPL）です。

[QOwnNotes - cross-platform open source plain-text file notepad](http://www.qownnotes.org/)
[pbek/QOwnNotes: QOwnNotes is a plain-text file notepad and todo-list manager with markdown support and ownCloud integration.](https://github.com/pbek/QOwnNotes)

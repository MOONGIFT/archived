海外出張などでUberを使わないことはほぼあり得なくなっています。Googleマップとも連携し、行きたい場所まですぐに連れて行ってくれるUberはとても便利です（各地で問題となっていますが、利用者としては最高の仕組みでしょう）。
そんなUberのAPIを使って作られているのがUber CLIです。閲覧のみですが面白い仕組みです。

## Uber CLIの使い方

timeサブコマンドでは目的地にUberが来る時間が分かります。

```
$ uber time '1church avenue 903, mascot'
┌────────────────────────────────────────────────┐
│ ? 903/1 Church Ave, Mascot NSW 2020, Australia │
├──────────────────┬─────────────────────────────┤
│        ⏳         │              ?              │
├──────────────────┼─────────────────────────────┤
│ 2 min.           │ uberX                       │
├──────────────────┼─────────────────────────────┤
│ 3 min.           │ UberBLACK,uberTAXI          │
├──────────────────┼─────────────────────────────┤
│ 4 min.           │ UberLUX                     │
├──────────────────┼─────────────────────────────┤
│ 7 min.           │ UberSUV                     │
└──────────────────┴─────────────────────────────┘
```

priceサブコマンドでは現在の場所と目的地を入れて見積もりを取れます。

```
$ uber price -s '1church avenue 903, mascot' -e 'Opera house, Sydney'
┌───────────┬───────────────────────┬──────────┬─────────┬──────────┐
│     ?     │           ?           │    ?     │    ⏳    │ ? Surge? │
├───────────┼───────────────────────┼──────────┼─────────┼──────────┤
│ uberX     │ $27-$34               │ 6.01 mi. │ 15 min. │ ?        │
├───────────┼───────────────────────┼──────────┼─────────┼──────────┤
│ UberBLACK │ $55-$69               │ 6.01 mi. │ 15 min. │ ?        │
├───────────┼───────────────────────┼──────────┼─────────┼──────────┤
│ UberLUX   │ $79-$98               │ 6.01 mi. │ 15 min. │ ?        │
├───────────┼───────────────────────┼──────────┼─────────┼──────────┤
│ UberSUV   │ $79-$98               │ 6.01 mi. │ 15 min. │ ?        │
├───────────┼───────────────────────┼──────────┼─────────┼──────────┤
│ uberTAXI  │ ?undefined-?undefined │ 6.01 mi. │ 15 min. │ ?        │
├───────────┼───────────────────────┴──────────┴─────────┴──────────┤
│     ?     │ 903/1 Church Ave, Mascot NSW 2020, Australia          │
├───────────┼───────────────────────────────────────────────────────┤
│     ?     │ Bennelong Point, Sydney NSW 2000, Australia           │
└───────────┴───────────────────────────────────────────────────────┘
```

UberがWeb APIを公開しているからこそ取れるものですが、住所を文字で入れるだけというのが手軽で面白いです。日本では規制がかかっているUberですが、海外のように普及してくれると旅行者にとってとても便利なものになることでしょう。

Uber CLIはnode/JavaScript製のオープンソース・ソフトウェア（MIT License）です。

GitHubは単なるコードリポジトリサービスではありません。よく使っているプロジェクトの動向をチェックしたり、気になる開発者をフォローしたりできます。スターはちょっとした応援になります。
そんなGitHubをiPhone/Androidの中で使えるGit Pointを紹介します。

## Git Pointの使い方

起動してGitHubアカウントでログインした画面です。フォローしているユーザ、リポジトリの情報が追えます。

![](http://images.moongift.jp/2017/08/open-source-git-point-3_thumb.d44061a33cbc2abc2926f73d91b4046f.png "http://images.moongift.jp/2017/08/open-source-git-point-3.d44061a33cbc2abc2926f73d91b4046f.png")

Issueの詳細を確認することもできます。

![](http://images.moongift.jp/2017/08/open-source-git-point-2_thumb.5f758ff8f8de87a96f2cd1e731059f1d.png "http://images.moongift.jp/2017/08/open-source-git-point-2.5f758ff8f8de87a96f2cd1e731059f1d.png")

プロフィール画面です。

![](http://images.moongift.jp/2017/08/open-source-git-point-1_thumb.aa6b455bd38b2175400fe8d1d243d1c1.png "http://images.moongift.jp/2017/08/open-source-git-point-1.aa6b455bd38b2175400fe8d1d243d1c1.png")

Git PointはReact Nativeで作られているのでAndroid、iOSのどちらでも確認できます。ソースコードの閲覧であったり、Issueを作成したりすることもできますのでわざわざWebブラウザで確認したりせずにGitHubを使いこなせるでしょう。

Git PointはReact Native/JavaScript製のオープンソース・ソフトウェア（MIT License）です。

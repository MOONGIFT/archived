GUIのWebブラウザの歴史は長いです。Mosaicの時代から考えるとすでに25年近いのではないでしょうか。その後NetscapeやFirefox、IE、Safari、Opera、Chromeなど様々なブラウザが誕生しました。
そんなブラウザの遍歴をうかがい知れるのがCSS Browsersです。なんとCSSで作られたブラウザウィンドウです。

## CSS Browsersの使い方

基本的なフレームです。これは特に何かのブラウザという訳ではないようです。

![](http://images.moongift.jp/2017/08/open-source-css%20browsers-3_thumb.7081ac240f928fe3068d54112c208a29.png "http://images.moongift.jp/2017/08/open-source-css%20browsers-3.7081ac240f928fe3068d54112c208a29.png")

Safari7風。

![](http://images.moongift.jp/2017/08/open-source-css%20browsers-2_thumb.4047259890d04a8383c5d27c95117b7f.png "http://images.moongift.jp/2017/08/open-source-css%20browsers-2.4047259890d04a8383c5d27c95117b7f.png")

Netscape Navigator風。この完成度がやけに高く感じられます。

![](http://images.moongift.jp/2017/08/open-source-css%20browsers-1_thumb.4df2d34be69c88015404c9a708a353a9.png "http://images.moongift.jp/2017/08/open-source-css%20browsers-1.4df2d34be69c88015404c9a708a353a9.png")

CSS Browsersはスタイルシートだけでかつての懐かしいブラウザウィンドウを再現しています。何かに役立つという訳ではない気がしますが、面白い試みです。画像を使わないというところにもこだわりが感じられます。

CSS BrowsersはCSS3製のオープンソース・ソフトウェア（ISC License）です。

[Superpencil/css-browsers: CSS-made browser windows to avoid using images](https://github.com/Superpencil/css-browsers)

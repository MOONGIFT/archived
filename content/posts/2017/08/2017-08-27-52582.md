レスポンシブWebデザインの広がりに合わせて利用が広がっているのがグリッドベースのデザインです。そんな中、ついに登場したのがCSS Gridになります。Web標準だけで手軽にグリッドデザインが実現できるようになりました。
そんなCSS Gridもまだまだ普及していない技術だけに手探り感があります。そこで使ってみたいのがCss Grid Highlighter for Chromeになります。

## Css Grid Highlighter for Chromeの使い方

Chrome機能拡張としてインストールします。

![](http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-6_thumb.45a07b372e9fa80e717a30fc65080cc6.png "http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-6.45a07b372e9fa80e717a30fc65080cc6.png")

そうするとグリッドの設定がハイライト表示されるようになります。

![](http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-5_thumb.43ed6c20e8b440cb467e6ec5fd4fa7bd.png "http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-5.43ed6c20e8b440cb467e6ec5fd4fa7bd.png")

複雑なデザインであっても（Aは横、Bは縦に二つ分です）分かりやすくなります。

![](http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-4_thumb.76df925d874ee2849ebc1a9eea499382.png "http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-4.76df925d874ee2849ebc1a9eea499382.png")

さらに複雑なグリッド構成。

![](http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-3_thumb.1b21a58f72a9ea5c5afdce6a2af00136.png "http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-3.1b21a58f72a9ea5c5afdce6a2af00136.png")

見た感じと実際の設定が異なる場合、ハイライト表示が役立ちます。

![](http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-2_thumb.a49d5839cdb26337a69d9bbc5d238295.png "http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-2.a49d5839cdb26337a69d9bbc5d238295.png")

グリッド同士に幅がある場合の表示です。

![](http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-1_thumb.1721e862c7bd2d386889a7f1326f30cd.png "http://images.moongift.jp/2017/08/open-source-css%20grid%20highlighter%20for%20chrome-1.1721e862c7bd2d386889a7f1326f30cd.png")

CSS Grid Highlighter for Chromeを使うことで、これからCSS Gridを学んだり、実際に使っていく際に役立ってくれることでしょう。まだ対応ブラウザは多くありませんが、今後普及していくテクニックだけに早めの習得がお勧めです。

CSS Grid Highlighter for ChromeはGoogle Chrome用のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

[ademilter/chrome-css-grid-highlighter: Css Grid Highlighter for Chrome](https://github.com/ademilter/chrome-css-grid-highlighter)

CSS3によって多くのデザインがCSSだけで実現できるようになっています。しかしまだまだできないことも存在します。その一つがフォントです。複雑でバランスが大事なフォントはWeb Fontを使って表現するのが基本です。
そんな中作られたのがCSS SANSです。CSSだけで作られた、「時代によって変わる」がコンセプトのフォントです。

## CSS SANSの使い方

例です。

![](http://images.moongift.jp/2017/08/open-source-css%20sans-4_thumb.2d6da6b138f312accbd9a9bb44580c90.png "http://images.moongift.jp/2017/08/open-source-css%20sans-4.2d6da6b138f312accbd9a9bb44580c90.png")

CSSなので一部だけハイライトさせるのも簡単です。

![](http://images.moongift.jp/2017/08/open-source-css%20sans-3_thumb.2fe8f11161a272d6156d8f8364a61623.png "http://images.moongift.jp/2017/08/open-source-css%20sans-3.2fe8f11161a272d6156d8f8364a61623.png")

BのカーブもCSSだけで表現されています。

![](http://images.moongift.jp/2017/08/open-source-css%20sans-2_thumb.e1ed350a3c40dfd25a92068204eedb2f.png "http://images.moongift.jp/2017/08/open-source-css%20sans-2.e1ed350a3c40dfd25a92068204eedb2f.png")

Zの横棒は一つのスタイル設定で作られています。

![](http://images.moongift.jp/2017/08/open-source-css%20sans-1_thumb.ad56d8d6f4d6fec86c2bdd568d5e8cf8.png "http://images.moongift.jp/2017/08/open-source-css%20sans-1.ad56d8d6f4d6fec86c2bdd568d5e8cf8.png")

CSS SANSはCSS1/2/3によってデザインが変わります。CSS1.0の時代では殆ど文字として認識できません。それが3.0になってフォントとして成り立つレベルになっています。そうやってCSS SANSはWebブラウザやインターネットの発展に合わせて進化するのです。これはWebブラウザの違いによるデザイン差異を防止したいというフォントの考え方とは全く逆と言えます。

CSS SANSはCSS3製のオープンソース・ソフトウェア（MIT License）です。

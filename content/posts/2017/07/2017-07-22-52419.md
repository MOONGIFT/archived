皆さんがGitHubのスターを何のためにつけていますか。応援として使っている方もいるでしょう。スターを付けているリポジトリを取得することもできるので、ブックマークとして使っている方もいます。ただ、あまり便利に使える訳ではありません。
そこで一緒に使っていきたいのがGithub Stars Managerです。GitHubでつけたスターを管理できるChrome機能拡張です。

## Github Stars Managerの使い方

スターを付けたリポジトリ一覧のところにタグ（現在はuntaggedのみ）が追加されます。

![](http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-4_thumb.b2a8ab2def1bde94a37f2a6253a015fb.png "http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-4.b2a8ab2def1bde94a37f2a6253a015fb.png")

タグを追加します。

![](http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-3_thumb.cf42508cdf0700a76f66351c991cd495.png "http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-3.cf42508cdf0700a76f66351c991cd495.png")

タグを付けていきます。

![](http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-2_thumb.a37697a0d30cd274f33a1bdfd3cefd13.png "http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-2.a37697a0d30cd274f33a1bdfd3cefd13.png")

タグを選択するとリポジトリを絞り込めます。

![](http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-1_thumb.36fe66c477c46f7b786fcaeeac8069d9.png "http://images.moongift.jp/2017/07/open-source-github%20stars%20manager-1.36fe66c477c46f7b786fcaeeac8069d9.png")

Github Stars Managerを使うことでスターを付けたリポジトリを便利に管理できるようになります。多数のスターを付けている方は特に、検索以外の絞り込みができると使いやすくなるでしょう。公式として提供して欲しいとすら思える機能ではないでしょうか。

Github Stars ManagerはGoogle Chrome機能拡張のオープンソース・ソフトウェア（MIT License）です。

[gabrielgodoy/github-stars-manager: Chrome extension that allows you to manage your Github stars with tags, and to create a bookmark folder with all your stars organized by the tags you created](https://github.com/gabrielgodoy/github-stars-manager)

人の目を引くデザインは奇抜なものばかりではありません。普段何気なく使っているものが別な場所にも出てきたりすると驚かれることもあるでしょう。特にコンピュータ内で自然に存在するものを再現すると興味を持つ人たちがたくさんいます。
今回紹介するWobble window jQuery pluginもそんなライブラリの一つです。柔らかなインタフェースを実現します。

## Wobble window jQuery pluginの使い方

デモです。柔らかくなります。

プルンプルンです。

![](http://images.moongift.jp/2017/05/open-source-wobble%20window%20jquery%20plugin-2.90cf4b814062b90d680e5df70882deae.gif "http://images.moongift.jp/2017/05/open-source-wobble%20window%20jquery%20plugin-2.90cf4b814062b90d680e5df70882deae.gif")

複数に適用する例。

![](http://images.moongift.jp/2017/05/open-source-wobble%20window%20jquery%20plugin-1.86aa2bfad3cb86e0aaeb9b581fd2dfcc.gif "http://images.moongift.jp/2017/05/open-source-wobble%20window%20jquery%20plugin-1.86aa2bfad3cb86e0aaeb9b581fd2dfcc.gif")

Wobble window jQuery pluginはCanvasを使って表現されています。そのため、滑らかにアニメーションできるようになっています。普段直線的で固いイメージのある枠線がプルプルと震えるのはなかなか面白いデザインではないでしょうか。

Wobble window jQuery pluginはjQuery/JavaScript製のオープンソース・ソフトウェア（MIT License）です。

[NiklasKnaack/jquery-wobblewindow-plugin: Wobble Window is a fancy jQuery plugin that helps you create a cool, interactive, configurable, HTML5 canvas based gooey effect when the mouse moves in/out a specified element.](https://github.com/NiklasKnaack/jquery-wobblewindow-plugin)

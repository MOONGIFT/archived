CSSの使い方をちゃんと覚えるというのは大変です。しかも日々進化しており、新しいプロパティが追加されたりします。もちろんすぐに汎用的に使える訳ではありませんが、先進的なブラウザからどんどん導入が進んでいきます。放置しておくとあっという間に時代に取り残されてしまうでしょう。
今回紹介するGrid GardenはCSS Grid Layout Moduleについてゲームを通じて学習できるソフトウェアです。

## Grid Gardenの使い方

クイズが出ていますので、それを達成できるようなCSSグリッドを記述します。

![](http://images.moongift.jp/2017/04/open-source-grid%20garden-3_thumb.82a53d05a13922bbbb2718e7aa3c1f78.png "http://images.moongift.jp/2017/04/open-source-grid%20garden-3.82a53d05a13922bbbb2718e7aa3c1f78.png")

負の数値もサポートしていることを示す例。

![](http://images.moongift.jp/2017/04/open-source-grid%20garden-2_thumb.a783c2c447d7ef3612290d1c48b0f162.png "http://images.moongift.jp/2017/04/open-source-grid%20garden-2.a783c2c447d7ef3612290d1c48b0f162.png")

イラストが可愛らしいですし、すぐに覚えられます。

![](http://images.moongift.jp/2017/04/open-source-grid%20garden-1_thumb.de4fc985606aa1321d89429ed1f4f614.png "http://images.moongift.jp/2017/04/open-source-grid%20garden-1.de4fc985606aa1321d89429ed1f4f614.png")

Grid Gardenではマス目に記述されたイラストをお題に合うように記述していきます。最初は分かりづらいかも知れませんが、慣れてくればすぐに書けるようになるでしょう。

Grid GardenはJavaScript/CSS3製のオープンソース・ソフトウェア（MIT License）です。

[Grid Garden - A game for learning CSS grid](http://cssgridgarden.com/)

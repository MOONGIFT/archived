最近ではクラウドにファイルを置くのが当たり前になっています。DropboxやBox、Google Driveなどを仕事、プライベートの両面で使っている人は多いのではないでしょうか。
そんな方であればCloud File Browserの良さに気づくのではないでしょうか。クラウドベースのファイルブラウザを紹介します。

## Cloud File Browserの使い方

![Dropboxとの接続例。普通にファイル、フォルダが見られます。](http://images.moongift.jp/2014/09/Screenshot%202014-09-22%2022.46.08_thumb.cd3041d33e1224526a1917d35bfb4a35.png "http://images.moongift.jp/2014/09/Screenshot%202014-09-22%2022.46.08.cd3041d33e1224526a1917d35bfb4a35.png")

![画像の場合はサムネイルも。ファイルはダウンロードでいます。](http://images.moongift.jp/2014/09/Screenshot%202014-09-22%2022.46.56_thumb.02806a0a16514ca1e038bb7e6efd2e1d.png "http://images.moongift.jp/2014/09/Screenshot%202014-09-22%2022.46.56.02806a0a16514ca1e038bb7e6efd2e1d.png")

![Google Driveも同じように表示できます。](http://images.moongift.jp/2014/09/Screenshot%202014-09-22%2022.47.30_thumb.3a350458f7734bfe38b57c9d8f28ee33.png "http://images.moongift.jp/2014/09/Screenshot%202014-09-22%2022.47.30.3a350458f7734bfe38b57c9d8f28ee33.png")

Cloud File Browserではファイルのアップロードとダウンロードが可能です。また、サービスとしてはBox/Dropbox/Google Drive/OneDriveに対応しています。ファイルの編集はできませんが、ファイルブラウザとして面白いソフトウェアです。

Cloud File Browserはnode/JavaScript製、Apache License 2.0のオープンソース・ソフトウェアです。

[Cloud Elements - Cloud File Browser](https://filebrowser.io/)

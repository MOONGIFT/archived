Webやアプリのデザインを行う際に、アテで文字列を適用します。同じ文字を繰り返す、小説の一文を使うなど様々な方法がありますが、むしろテキストでない方がいいこともあるでしょう。
そんな時に使ってみたいのがReact Spectrumです。カラフルなプレースホルダーを生成します。

## React Spectrumの使い方

生成例です。

![](https://images.moongift.jp/2020/01/open-source-react%20spectrum-3_thumb.ccd5fe27a6fbc24a0517c477a57cdd0f.jpg "https://images.moongift.jp/2020/01/open-source-react%20spectrum-3.ccd5fe27a6fbc24a0517c477a57cdd0f.png")

太さも色も自在に変えられます。

![](https://images.moongift.jp/2020/01/open-source-react%20spectrum-2_thumb.8a819e57e5bb8ea78b8020cc8df779c0.jpg "https://images.moongift.jp/2020/01/open-source-react%20spectrum-2.8a819e57e5bb8ea78b8020cc8df779c0.png")

四角いパターンも。

![](https://images.moongift.jp/2020/01/open-source-react%20spectrum-1_thumb.8ed87a86c5c1541e8d9351336bf76114.jpg "https://images.moongift.jp/2020/01/open-source-react%20spectrum-1.8ed87a86c5c1541e8d9351336bf76114.png")

ダークモードにも対応しています。

![](https://images.moongift.jp/2020/01/open-source-react%20spectrum-5_thumb.d88b9c69672e4e275fe90a895298ed86.jpg "https://images.moongift.jp/2020/01/open-source-react%20spectrum-5.d88b9c69672e4e275fe90a895298ed86.png")

React Spectrumを使うことで、ユーザに雰囲気を伝えることができるでしょう。React SpectrumはReactで生成していますので、React以外のWebアプリであれば生成したプレースホルダーを画像として使うのも良さそうです。

React SpectrumはTypeScript製のオープンソース・ソフトウェア（MIT License）です。

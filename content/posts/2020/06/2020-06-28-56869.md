Webブラウジングをしていて英文などを翻訳したいと思ったことはないでしょうか。よくあるのは別なタブで翻訳サイトを開いて、そこにテキストを貼り付けるという操作です。クリップボードにコピーしたりして、意外と操作が多かったりします。
そこで使ってみたいのが‎Translate Web for Safariです。翻訳サイトへの一発ジャンプを可能にするSafari機能拡張です。

## ‎Translate Web for Safariの使い方

設定画面です。テキストを選択した場合としていない場合で、ジャンプするサイトを変更できます。

![](https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-4_thumb.86fb37540824d8afc6a7e12ccf838759.jpg "https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-4.86fb37540824d8afc6a7e12ccf838759.png")

コンテクストメニューに「〜を翻訳」が追加されます。

![](https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-3_thumb.b9326555e964fffd133d9549c99b35d5.jpg "https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-3.b9326555e964fffd133d9549c99b35d5.png")

例えばテキストを選択していた場合はDeepLにしたり。

![](https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-2_thumb.215645de1bb8ae9e47a61ff78196db2c.jpg "https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-2.215645de1bb8ae9e47a61ff78196db2c.png")

ページ全体の場合はGoogleまたはBingが指定できます。

![](https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-1_thumb.d74fb7546ee4fc822ede797a416e6318.jpg "https://images.moongift.jp/2020/06/open-source-translate%20web%20for%20safari-1.d74fb7546ee4fc822ede797a416e6318.png")

‎Translate Web for SafariはGoogle Chromeなどでは機能拡張として提供されている類似ソフトウェアのSafari版です。テキスト選択時か否かで飛び先が変更できるのが便利です。Safari派の方はぜひ使ってみましょう。

‎Translate Web for SafariはSwift製のオープンソース・ソフトウェア（MIT License）です。

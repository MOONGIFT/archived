機械学習などで重要なのは、学習元になるデータを用意することです。元になるデータが十分になければ適切な結果は返ってきませんので、そのためにデータをラベリングする必要があります。例えば猫、テキストなどの部分にラベルを付けます。
今回紹介するUniversal Data Toolは、そんなラベリングを行うための専用ツールです。

## Universal Data Toolの使い方

起動しました。

![](https://images.moongift.jp/2020/03/open-source-universal-data-tool-5_thumb.464c1a83dd52508ecaeebc570ec3c197.jpg "https://images.moongift.jp/2020/03/open-source-universal-data-tool-5.464c1a83dd52508ecaeebc570ec3c197.png")

画像に対してラベルを付けます。

![](https://images.moongift.jp/2020/03/open-source-universal-data-tool-4_thumb.f4b10a1578d59e889bfbbfebf704e89f.jpg "https://images.moongift.jp/2020/03/open-source-universal-data-tool-4.f4b10a1578d59e889bfbbfebf704e89f.png")

別な画像には別なラベルを。

![](https://images.moongift.jp/2020/03/open-source-universal-data-tool-3_thumb.2ad9fd57ba00d1460ccf4ce794310434.jpg "https://images.moongift.jp/2020/03/open-source-universal-data-tool-3.2ad9fd57ba00d1460ccf4ce794310434.png")

猫を囲んだりできます。

![](https://images.moongift.jp/2020/03/open-source-universal-data-tool-2_thumb.edd37a8366306078dc73c218f4ca650d.jpg "https://images.moongift.jp/2020/03/open-source-universal-data-tool-2.edd37a8366306078dc73c218f4ca650d.png")

設定です。

![](https://images.moongift.jp/2020/03/open-source-universal-data-tool-1_thumb.14da6cd779d31ed46469b04dc040b2a0.jpg "https://images.moongift.jp/2020/03/open-source-universal-data-tool-1.14da6cd779d31ed46469b04dc040b2a0.png")

Universal Data Toolは画像に対してポイントで印をつけたり、四角で囲む、さらに多角形で囲むといった操作ができます。画像だけでなく、テキストやドキュメントなどについてもラベリングが可能とのことです。

Universal Data ToolはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

[UniversalDataTool/universal-data-tool: Collaborate &amp; label any type of data, images, text, or documents, in an easy web interface or desktop app.](https://github.com/UniversalDataTool/universal-data-tool)

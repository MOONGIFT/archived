物を作っていくゲームはなんと面白いことでしょう。一気に大きなものを作るのではなく、小さなものを積み重ねていくのに楽しさを感じます。シムシティはもちろん、Factorioのような工場建設ゲームもそうです。
今回紹介するshapez.ioはFactorioにインスパイアされた工場製造ゲームです。

## shapez.ioの使い方

ゲーム開始です。

![](https://images.moongift.jp/2020/06/open-source-shapez.io-6_thumb.be5033ee2923354e43ecae8ebf63a3e1.jpg "https://images.moongift.jp/2020/06/open-source-shapez.io-6.be5033ee2923354e43ecae8ebf63a3e1.png")

抽出器を作成します。

![](https://images.moongift.jp/2020/06/open-source-shapez.io-5_thumb.63a7f68f771c987e1e07237ac903d20b.jpg "https://images.moongift.jp/2020/06/open-source-shapez.io-5.63a7f68f771c987e1e07237ac903d20b.png")

抽出器からベルトコンベアを使ってハブまでアイテムを輸送します。

![](https://images.moongift.jp/2020/06/open-source-shapez.io-4_thumb.944dacc938e89daf408cf2eda2123b7d.jpg "https://images.moongift.jp/2020/06/open-source-shapez.io-4.944dacc938e89daf408cf2eda2123b7d.png")

レベルが上がると製造できるものが増えていきます。

![](https://images.moongift.jp/2020/06/open-source-shapez.io-3_thumb.5f866ebf479da67ed5b497c308afd47e.jpg "https://images.moongift.jp/2020/06/open-source-shapez.io-3.5f866ebf479da67ed5b497c308afd47e.png")

設定です。日本語にも対応しています。

![](https://images.moongift.jp/2020/06/open-source-shapez.io-2_thumb.f7feec1b22877e2600dd6523cd09435e.jpg "https://images.moongift.jp/2020/06/open-source-shapez.io-2.f7feec1b22877e2600dd6523cd09435e.png")

どんどん拡張していきます。

![](https://images.moongift.jp/2020/06/open-source-shapez.io-1_thumb.0f19cb0dcd0861fc90c0d310cfad2e9b.jpg "https://images.moongift.jp/2020/06/open-source-shapez.io-1.0f19cb0dcd0861fc90c0d310cfad2e9b.png")

shapez.ioは初期状態ではできることは多くありませんが、徐々に設置できるアイテムが増えたり、複雑な操作が必要になってきます。Webで公開されているのがデモ版で、正式版はSteamからゲットできます。

shapez.ioはJavaScript製のオープンソース・ソフトウェア（GPL v3）です。

[shapez.io - Build automated factories to build, combine and color shapes!](https://shapez.io/)
[tobspr/shapez.io: shapez.io is an open source base building game inspired by factorio! Available on web &amp; desktop](https://github.com/tobspr/shapez.io)

GitHubを通じて開発者とコミュニケーションしている人は多いでしょう。社内においてもIssueやPRを使って対話したり、意見を求められたりします。必ずしもコードを書かずともGitHubを使っているという人は多いはずです。
そうした方にお勧めなのがGitTouchです。Flutterで作られたGitHubクライアントアプリです。

## GitTouchの使い方

まずアカウントを登録します。これはOAuth2認証で行います。

![](https://images.moongift.jp/2019/02/open-source-gittouch-8_thumb.7aa8371b70fc1b2a48e2d4f450664637.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-8.7aa8371b70fc1b2a48e2d4f450664637.png")

アカウントを追加しました。

![](https://images.moongift.jp/2019/02/open-source-gittouch-7_thumb.02c3723fc1472696dc0e507a7ae30e80.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-7.02c3723fc1472696dc0e507a7ae30e80.png")

ニュースでアクティビティを確認できます。

![](https://images.moongift.jp/2019/02/open-source-gittouch-6_thumb.46ba93df0310dcb3f5bf697598a4dbc3.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-6.46ba93df0310dcb3f5bf697598a4dbc3.png")

通知一覧。

![](https://images.moongift.jp/2019/02/open-source-gittouch-5_thumb.53a8f580fe03258a5d05d0c9c32ce270.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-5.53a8f580fe03258a5d05d0c9c32ce270.png")

リポジトリ検索機能もあります。

![](https://images.moongift.jp/2019/02/open-source-gittouch-4_thumb.6a22d45af851ce2e648952cccdef8ed9.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-4.6a22d45af851ce2e648952cccdef8ed9.png")

個人のプロフィール画面。

![](https://images.moongift.jp/2019/02/open-source-gittouch-3_thumb.d1455121c378806b0348a5915f84e8ed.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-3.d1455121c378806b0348a5915f84e8ed.png")

テーマを変更しました。

![](https://images.moongift.jp/2019/02/open-source-gittouch-1_thumb.2e200b0f0dfe28f3cd00ccaea77866d9.jpg "https://images.moongift.jp/2019/02/open-source-gittouch-1.2e200b0f0dfe28f3cd00ccaea77866d9.png")

GitTouchはGitHubの情報を確認したり、コメントしたりと言ったGitHub上での活動を支援してくれます。外出時にスマートフォンから操作するよりも手軽に対応できるでしょう。

GitTouchはDart/Flutter製のオープンソース・ソフトウェア（MIT License）です。

[pd4d10/git-touch: Open source GitHub client built with Flutter](https://github.com/pd4d10/git-touch)

GUIのOSであれば当然のように入っているのが計算機です。Googleで計算できるようになって以来、あまり使うこともなくなりましたが、それでもネットワークに繋がっていない時には便利です。
そんな計算機、Windows10に付属しているWindows Calculatorがオープンソース化されました。なぜ計算機が…という気もしなくもないですが面白いですね。

## Windows Calculatorの使い方

Windows 10の計算機は使ったことがなかったのですが、思いの外大きいです。

![](https://images.moongift.jp/2019/03/open-source-windows%20calculator-7_thumb.b1339c194d8e28e2365b8c3b15659b08.jpg "https://images.moongift.jp/2019/03/open-source-windows%20calculator-7.b1339c194d8e28e2365b8c3b15659b08.png")

通常の計算機だけではなく、関数電卓や通貨計算、温度や重さなど様々な計算機能が揃っています。

![](https://images.moongift.jp/2019/03/open-source-windows%20calculator-6_thumb.7082513c906e316f3153f6a9268e3983.jpg "https://images.moongift.jp/2019/03/open-source-windows%20calculator-6.7082513c906e316f3153f6a9268e3983.png")

計算結果は履歴が残ります。

![](https://images.moongift.jp/2019/03/open-source-windows%20calculator-5_thumb.e9ce3438b32358836318bb1c6dd7f20d.jpg "https://images.moongift.jp/2019/03/open-source-windows%20calculator-5.e9ce3438b32358836318bb1c6dd7f20d.png")

通貨計算の画面。

![](https://images.moongift.jp/2019/03/open-source-windows%20calculator-4_thumb.ee4ccbf840b8a9fe5bc1211433900f89.jpg "https://images.moongift.jp/2019/03/open-source-windows%20calculator-4.ee4ccbf840b8a9fe5bc1211433900f89.png")

熱量計算の画面。

![](https://images.moongift.jp/2019/03/open-source-windows%20calculator-3_thumb.845f76abb239246e40472c55ff262406.jpg "https://images.moongift.jp/2019/03/open-source-windows%20calculator-3.845f76abb239246e40472c55ff262406.png")

プログラマー。16進数とかに変換できます。

![](https://images.moongift.jp/2019/03/open-source-windows%20calculator-2_thumb.06a564a9e95fc489c566f1f0f5fee37a.jpg "https://images.moongift.jp/2019/03/open-source-windows%20calculator-2.06a564a9e95fc489c566f1f0f5fee37a.png")

Windows Calculatorは他のOSの計算機に比べて圧倒的に多機能で、そして大きいアプリです。ここまで大きくなくとも…と思いますが、タブレットなどではこのくらいのサイズでないといけないのかも知れません。そんなWindows Calculator、オープンソース化されたことで新しい機能が外部から追加される日が来るのでしょうか。

Windows CalculatorはC++製のオープンソース・ソフトウェア（MIT License）です。

[Microsoft/calculator: Windows Calculator: A simple yet powerful calculator that ships with Windows](https://github.com/Microsoft/calculator)

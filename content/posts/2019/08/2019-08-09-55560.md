オープンソースに対して何らかの活動を行うことが開発者としてもポートフォリオとして重要になっています。転職などを行う際にGitHubのアカウントをチェックするのも当然になっています。
今回紹介するGPortfolioはGitHubでの活動をベースにしたポートフォリオ生成ソフトウェアです。

## GPortfolioの使い方

ポートフォリオの例です。

![](https://images.moongift.jp/2019/07/open-source-gportfolio_thumb.606c4bef6114475fa9de50d2518e752b.jpg "https://images.moongift.jp/2019/07/open-source-gportfolio.606c4bef6114475fa9de50d2518e752b.png")

GPortfolioが生成するのは個人のアカウントと画像、自分が持っているリポジトリの一覧となっています。リポジトリ名をクリックすると、各リポジトリのURLに飛ぶようになっています。GitHubの画一的な一覧ではなく、ちょっとしたカスタマイズでぐっと見やすくなるでしょう。

GPortfolioはnode/JavaScript製のオープンソース・ソフトウェア（MIT License）です。

[GPortfolio/GPortfolio: Creating an automatic portfolio based on Github profile](https://github.com/GPortfolio/GPortfolio)

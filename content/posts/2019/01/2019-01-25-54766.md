Gitコマンドをターミナル上で使っている人は多いはずです。addやcommit、mergeなどであれば問題ありませんが、普段使わない操作を行おうと思った時にコマンドが分からずにWeb検索している人も多いでしょう。それだけGitは機能が豊富で、パラメータの順番まで決まっているので覚えるのが大変です。
そこで使ってみたいのがGit Explorerです。Gitのコマンドについて、ウィザード方式で教えてくれます。

## Git Explorerの使い方

トップページです。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-7_thumb.a646519277f77ee5162d1303bed96918.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-7.a646519277f77ee5162d1303bed96918.png")

何をしたいかを選択します。サブコマンドが並んでいます。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-6_thumb.bea317d0d1551a6ac702347d1d35f315.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-6.bea317d0d1551a6ac702347d1d35f315.png")

そしてその後の選択肢も表示されます。この内容はサブコマンドによって異なります。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-5_thumb.009fe76eb9f9c9ffdd82d353724edf53.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-5.009fe76eb9f9c9ffdd82d353724edf53.png")

ウィザードが完了すると右側にコマンドが表示されます。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-4_thumb.85361c292cf8fd6ec2c7310c04a07f1d.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-4.85361c292cf8fd6ec2c7310c04a07f1d.png")

別なコマンドを選んだ例。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-3_thumb.dc33e6aad0a5c62f5a6c10b511187ad1.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-3.dc33e6aad0a5c62f5a6c10b511187ad1.png")

さらに説明文が表示される例も。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-2_thumb.20657260d60f1dde29054713a68a5cf3.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-2.20657260d60f1dde29054713a68a5cf3.png")

ダークモードも用意されています。

![](https://images.moongift.jp/2019/01/open-source-git%20explorer-1_thumb.de69ff39b460e067c75d971530c53cbe.jpg "https://images.moongift.jp/2019/01/open-source-git%20explorer-1.de69ff39b460e067c75d971530c53cbe.png")

Git Explorerは何をしたいかが分かっている人向けなので、Gitの基本的な使い方が分かっている人でないと使うのが難しいでしょう。ある程度分かっている人で、コマンドをど忘れしたり、パラメータの順番を確認したいと言った時に使えるサービスです。

Git ExplorerはJavaScript製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

懐古主義という言葉があります。過去の出来事は美化される傾向にあり、当時味わったはずの嫌な思い出は忘却の彼方へ飛んでいってしまいます。その結果、古き良きなんて言葉が出てきます。
今回紹介するWindows 95 UI Kitはその名の通り、Windows 95風のUIをWebブラウザ上に再現できるCSSデザインです。

## Windows 95 UI Kitの使い方

ボタン。業務システムっぽさがあります。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-7_thumb.57ac2ed228be4886282a7e668c5d81dc.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-7.57ac2ed228be4886282a7e668c5d81dc.png")

フォーム。デザインがシンプルです。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-6_thumb.41ff065dd6bb6b6ef01b9d4e67667f4c.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-6.41ff065dd6bb6b6ef01b9d4e67667f4c.png")

カード。懐かしいダイアログです。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-5_thumb.6b3f6af1d3606b5d0d5bf1d87b02e2bf.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-5.6b3f6af1d3606b5d0d5bf1d87b02e2bf.png")

ナビゲーションバー。いわゆるステータスバーですね。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-4_thumb.3bc4571e5d50aa5a7126e9e7f8e2e902.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-4.3bc4571e5d50aa5a7126e9e7f8e2e902.png")

タブ。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-3_thumb.1d5a7a6534f9c52a6107438e5754e95b.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-3.1d5a7a6534f9c52a6107438e5754e95b.png")

ツールチップ。実に無骨です。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-2_thumb.95debb76454d4c88a5e30372457916eb.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-2.95debb76454d4c88a5e30372457916eb.png")

タイポグラフィ。カクカクですね。

![](https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-1_thumb.62ba90644257a41f5442b5167850feec.jpg "https://images.moongift.jp/2019/09/open-source-windows%2095%20ui%20kit-1.62ba90644257a41f5442b5167850feec.png")

Windows 95 UI Kitはあの頃の懐かしいWindows 95のデザインを今に感じさせます。しかもBootstrap4製なので、汎用性もあります。今から思えば、このデザインに耐えられたことも不思議でしょう。昔風のWebサイトを作る際に役立ちそうです。

Windows 95 UI KitはCSS製のオープンソース・ソフトウェア（MIT License）です。

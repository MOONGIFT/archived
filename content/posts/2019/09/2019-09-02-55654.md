Windowsで標準インストールされているテキストエディタと言えばメモ帳（Notepad）です。あまりにも貧弱な機能で、殆ど使われることはないと思いますが、思いがけず利用することもあります。
そんなメモ帳を現実的に使えるものにしようと考えている（恐らく）のがNotepad3です。

## Notepad3の使い方

UIはメモ帳＋αくらいな感じに見えます。

![](https://images.moongift.jp/2019/08/open-source-notepad3-4_thumb.c977bb01e1a4c801f9dfa0961fcaa67e.jpg "https://images.moongift.jp/2019/08/open-source-notepad3-4.c977bb01e1a4c801f9dfa0961fcaa67e.png")

機能は多彩です。

![](https://images.moongift.jp/2019/08/open-source-notepad3-3_thumb.48e29ca4cc5427f916e7713f8f0a7ff2.jpg "https://images.moongift.jp/2019/08/open-source-notepad3-3.48e29ca4cc5427f916e7713f8f0a7ff2.png")

ダークモードにも対応しています。

![](https://images.moongift.jp/2019/08/open-source-notepad3-2_thumb.73a1c778a5a9ce40a468d0028aee997d.jpg "https://images.moongift.jp/2019/08/open-source-notepad3-2.73a1c778a5a9ce40a468d0028aee997d.png")

プログラミング言語のシンタックスハイライト、さらに折りたたみも可能です。

![](https://images.moongift.jp/2019/08/open-source-notepad3-1_thumb.e3cdb2c68f8bd53e6f7b00eb0f0b65d6.jpg "https://images.moongift.jp/2019/08/open-source-notepad3-1.e3cdb2c68f8bd53e6f7b00eb0f0b65d6.png")

Notepad3は文字コードにも数多く対応していますし、置換も使えるので、ある程度のプログラミングも可能です。標準のメモ帳もこれくらいの機能があれば…と思わずにはいられないでしょう。

Notepad3はWindows用のオープンソース・ソフトウェア（BSD）です。

[rizonesoft/Notepad3: Notepad like text editor based on the Scintilla source code. Notepad3 based on code from Notepad2 and MiniPath on code from metapath. Download Notepad3:](https://github.com/rizonesoft/Notepad3)

Googleカレンダーを自分のスケジュール管理に使っている人は多いかと思います。そして、Webサービス提供側としても日程情報が決まったら、それをGoogleカレンダーに登録するリンクを表示したいと思うことでしょう。
今回紹介するGoogle Calendar event link generatorはまさにその目的にぴったりなソフトウェアです。Googleカレンダー用のリンクを生成してくれます。

## Google Calendar event link generatorの使い方

カレンダーに登録する情報を入力します。

![](https://images.moongift.jp/2019/11/open-source-google%20calendar%20event%20link%20generator-3_thumb.b760eaf64460a5aa9493dd2a4d5b1211.jpg "https://images.moongift.jp/2019/11/open-source-google%20calendar%20event%20link%20generator-3.b760eaf64460a5aa9493dd2a4d5b1211.png")

リンクを生成しました。

![](https://images.moongift.jp/2019/11/open-source-google%20calendar%20event%20link%20generator-2_thumb.c19d842bbac24f1f10bc18b2ee845330.jpg "https://images.moongift.jp/2019/11/open-source-google%20calendar%20event%20link%20generator-2.c19d842bbac24f1f10bc18b2ee845330.png")

そのURLにアクセスすると、必要な情報が入力された状態で表示されます。

![](https://images.moongift.jp/2019/11/open-source-google%20calendar%20event%20link%20generator-1_thumb.a0a2588cc36e3846005642e542d3a1a4.jpg "https://images.moongift.jp/2019/11/open-source-google%20calendar%20event%20link%20generator-1.a0a2588cc36e3846005642e542d3a1a4.png")

Google Calendar event link generatorは自動でスケジュールを登録してくれる訳ではありませんが、後は保存ボタンを押すだけの状態にしてくれます。イベント系のサービスにあると便利なソフトウェアです。

Google Calendar event link generatorはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

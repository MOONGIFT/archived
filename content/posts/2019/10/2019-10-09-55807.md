オーディオのビジュアライザーをずっと見ていても飽きないという人は多いでしょう。ただ音楽を流すだけでなく、可視化することで別な楽しさが生まれます。そのため、幾つものパターンが作られています。
今回紹介するWeb Radio VisualizerはWebラジオを音源としたビジュアライザーです。

## Web Radio Visualizerの使い方

時間によって可視化方法が随時切り替わります。

![](https://images.moongift.jp/2019/09/open-source-web%20radio%20visualizer-2.08f2b586caa4218577f894d45295b8d1.gif "https://images.moongift.jp/2019/09/open-source-web%20radio%20visualizer-2.08f2b586caa4218577f894d45295b8d1.gif")

こんな円形のものも。

![](https://images.moongift.jp/2019/09/open-source-web%20radio%20visualizer-1.8917860262f16633bdf254cac13ee7c3.gif "https://images.moongift.jp/2019/09/open-source-web%20radio%20visualizer-1.8917860262f16633bdf254cac13ee7c3.gif")

Web Radio Visualizerはアニメーション部分だけを自作することもできます。もっと格好良いビジュアライズがあれば、Web Radio Visualizerに参加するのも良いでしょう。シンプルさを売りにしていますので、使いこなすのは難しくなさそうです。

Web Radio VisualizerはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

[victorqribeiro/radio: A simple javascript web radio visualizer](https://github.com/victorqribeiro/radio)

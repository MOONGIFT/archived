Windows 95が発売してからすでに25年が経とうとしている事実。4半世紀前になるのです。その頃からMac派なのでWindows 95発売日の祭りはニュースで見た程度なのですが、PCの歴史が変わった瞬間でもありました。
そんなWindows 95のUIを現在の技術、Reactで再現しようというのがReact95です。

## React95の使い方

トップページからしてこれです。このボロボロなビニール、CDあるあるですね。

![](https://images.moongift.jp/2019/04/open-source-react95-1_thumb.79a70acef5c33127671471b1c6d75845.jpg "https://images.moongift.jp/2019/04/open-source-react95-1.79a70acef5c33127671471b1c6d75845.png")

回る砂時計とアニメーションGIF。そして古くさい（褒め言葉）ボタン。

![](https://images.moongift.jp/2019/04/open-source-react95-4_thumb.335a47055288dfc1b17ff93594c901eb.jpg "https://images.moongift.jp/2019/04/open-source-react95-4.335a47055288dfc1b17ff93594c901eb.png")

カレンダー。懐かしいUIです。

![](https://images.moongift.jp/2019/04/open-source-react95-3_thumb.c2b19f665c2ebdc3b7b457d51a2eb839.jpg "https://images.moongift.jp/2019/04/open-source-react95-3.c2b19f665c2ebdc3b7b457d51a2eb839.png")

ちゃんと日付も選択できます。

![](https://images.moongift.jp/2019/04/open-source-react95-2_thumb.7a9e55851210c41312b3f690e7444728.jpg "https://images.moongift.jp/2019/04/open-source-react95-2.7a9e55851210c41312b3f690e7444728.png")

デフォルトで表示されている1995年08月24日はWindows95が発売した日です。ただし日本語版ではなく、英語版となっています（[日本語版は11月](https://ja.wikipedia.org/wiki/Microsoft_Windows_95)だそうです）。使いどころはそうそう多くないと思いますが、以前[メルカリで90年代当時のUIで再現したWebサイト](https://mercari-special.jp/~natsukashi/)が作られました。ああいったジョークサイトでも使えそうです。

React95はJavaScript製のオープンソース・ソフトウェア（MIT License）です。

[Refreshed Windows95 UI components for modern web apps - React95](https://react95.io/#launched)

GitHub for MacはGitHubを使った開発者用のGithubクライアント。

  

GitHub for MacはMac OSX用のフリーウェア。Githubの魅力に取り付かれてしまうとなかなか抜け出すことはできない。オープンソースを開発するときはもちろん、クローズドソースであってもGithub上で管理しているととても簡単で便利だ。

  

[![](https://images.moongift.jp/2011/06/ScreenShot2011-06-24-15.37.39_thumb.png)](https://images.moongift.jp/2011/06/a9db6c020c26ed823e58751d241939ae.png)  
**メイン画面**

  

単なるGitリポジトリサービスというよりも、開発者が集まっているだけにGithub自身がどんどん進化しているのが面白い。その最新作としてのGitHub for Macを紹介しよう。

  
<!--more-->  

GitHub for MacはGithubを管理するための専用クライアントアプリだ。これまでGithub向けのソフトウェアとしてはソーシャル性を重視した楽しめるソフトウェアが多かったが、GitHub for Macは完全に作業を行うための機能が詰まっている。

  

[![](https://images.moongift.jp/2011/06/ScreenShot2011-06-24-15.37.44_thumb.png)](https://images.moongift.jp/2011/06/37c9ff3e91ec7598f5f5c2d7b5800104.png)  
**ログ**

  

リポジトリに入って履歴を見たり、そのコミット内容を確認する、リポジトリのクローンを作成する、ブランチを同期したりマージする、.gitignoreを修正する、更新内容を確認してコミットするなどGitに関係する全ての作業がGitHub for Mac上で行える。GitHubでリポジトリを管理しているMac OSXユーザには必携のソフトウェアだろう。

**MOONGIFTはこう見る**

  

Gitの面白さは人それぞれだろうが、個人的にはどんどんコミットを行っていたり、ブランチを切ったりできるのが魅力だと思っている。Subversionでは下手にバッティングしないようにルールを明確にする必要があったが、Gitの場合は個人のリポジトリで好き勝手にできるイメージだ（もちろんルールは多少あった方がいいだろうが）。

  

そんな素早いサイクルの開発をGitHub for Macはサポートしてくれる。またクローンするのもクリック一つでできたりと、GitHubをどんどん活用していくために必要な機能が備わっている。これを使えばさらに開発がスムーズになるのではないだろうか。なお動作は64bitに限られるようだ。

アメリカでメッセージングアプリと言えばWhatsAppが一番シェアを持っているでしょう。日本では様相が異なるので情報が届きづらいですが、アメリカ人の知り合いが多い人は使っているかも知れません。
そんなWhatsApp風のUIをPWA（Progressive Web App）で再現したのがWhatsApp Clone PWAです。

## WhatsApp Clone PWAの使い方

最初の画面です。Googleアカウントで認証します。

![](https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-6_thumb.21f5fea6946f4a2fae4bd5c7ca384a36.jpg "https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-6.21f5fea6946f4a2fae4bd5c7ca384a36.png")

Roomsから人を選ぶとチャットできます。

![](https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-4_thumb.a329e73f475e92701a17c7a1f0157bef.jpg "https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-4.a329e73f475e92701a17c7a1f0157bef.png")

チャット履歴も確認できます。

![](https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-2_thumb.e2f8f5e8a623accc555cfad118a62188.jpg "https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-2.e2f8f5e8a623accc555cfad118a62188.png")

画像も送れます。

![](https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-1_thumb.c7748f4e074b0071e0f71d77ea44f998.jpg "https://images.moongift.jp/2021/04/open-source-whatsapp%20clone%20pwa-1.c7748f4e074b0071e0f71d77ea44f998.png")

WhatsApp Clone PWAはPWAであり、アプリとしてインストールできます。インストールしてしまえば、まるでWhatsAppクライアントのようです。Firebaseを使っておりバックエンドも用意できているので、独自のメッセージサービスを立ち上げる際の参考になりそうです。

WhatsApp Clone PWAはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

[WhatsApp Clone PWA](https://whatsappy-app.web.app/)
[aladinyo/WhatsappyApp: A whatsapp clone progressive web application built with react nodejs and firebase](https://github.com/aladinyo/WhatsappyApp)

WebはもはやOSといっても過言ではありません。多くのアプリケーションが動作し、情報が保存されています。しかしOSとは雰囲気が異なるので、使い方も大きく違います。垣根を越えるためにはOS風のUIが必要です。
そんな可能性が感じさせてくれるのがCSS Windows 10です。Windows 10風のUIをCSS/HTMLで再現しています。

## CSS Windows 10の使い方

まさにWindows 10のようです。

![](https://images.moongift.jp/2021/03/open-source-css%20windows%2010-2_thumb.f6ff26c4db71ddc0913073535123522b.jpg "https://images.moongift.jp/2021/03/open-source-css%20windows%2010-2.f6ff26c4db71ddc0913073535123522b.png")

ウィンドウをドラッグして移動できます。

![](https://images.moongift.jp/2021/03/open-source-css%20windows%2010-1.7415a8d174b2cd6e6e9162da74e63d5a.gif "https://images.moongift.jp/2021/03/open-source-css%20windows%2010-1.7415a8d174b2cd6e6e9162da74e63d5a.gif")

CSS Windows 10の機能はウィンドウをドラッグして移動できるのみで、スタートメニューを開いたり、アプリケーションを立ち上げたりはできません。その点は今後に期待したいところです。Webブラウザ上でOS風のUIが動けば、さらにWebのOS化が進みそうです。

CSS Windows 10はHTML/CSS製のソフトウェア（ライセンスはWTFPL）です。

[jianzhongli/csswin10: A CSS / JS based Windows 10 UI](https://github.com/jianzhongli/csswin10)

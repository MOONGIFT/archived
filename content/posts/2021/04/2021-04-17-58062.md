フロントエンド開発の便利な点は、Webブラウザじ実装されているJavaScriptエンジンでコードをすぐに実行、確認できることでしょう。HTMLやCSSをデバッグすることもでき、開発がさくさくと進められます。
そんなフロントエンド開発時にちょっとしたプレイグラウンドが欲しかったらCodeistを使ってみましょう。

## Codeistの使い方

左側にエディタ、右側にプレビューが表示されます。

![](https://images.moongift.jp/2021/03/open-source-codeist-4_thumb.2cc2963c92904dbe67d6eeb29627e186.jpg "https://images.moongift.jp/2021/03/open-source-codeist-4.2cc2963c92904dbe67d6eeb29627e186.png")

入力内容が即座にプレビューへ反映されます。

![](https://images.moongift.jp/2021/03/open-source-codeist-3_thumb.6a0440f7efa6a48ea24f191d4e8a99d8.jpg "https://images.moongift.jp/2021/03/open-source-codeist-3.6a0440f7efa6a48ea24f191d4e8a99d8.png")

CSSも同様です。

![](https://images.moongift.jp/2021/03/open-source-codeist-2_thumb.926e75de71db457c32690aab623925a6.jpg "https://images.moongift.jp/2021/03/open-source-codeist-2.926e75de71db457c32690aab623925a6.png")

JavaScriptも反映されます。

![](https://images.moongift.jp/2021/03/open-source-codeist-1_thumb.d534453397c0a08edb8d3bbf475a5bab.jpg "https://images.moongift.jp/2021/03/open-source-codeist-1.d534453397c0a08edb8d3bbf475a5bab.png")

JavaScriptも入力する毎に反映されているためか、若干入力が重たくなります。HTMLやCSSは大丈夫です。入力内容が保存できないのが残念ですが、スニペットを試すといった時には十分でしょう。

CodeistはJavaScript製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

[Codeist - IDE](https://codeist.netlify.app/)
[xKaran/Codeist: A web based code editor for rapid web development.](https://github.com/xKaran/Codeist)

アイコンセットはいくらあっても便利です。昔は小さな画像の集まり、その後はCSSスプライト、そして最近ではアイコンフォントを使うパターンが多くなっています。アイコンフォントは便利ですが、読み込みに失敗するとトウフだらけになってしまうのが難点です。
そこで使ってみたいのがCSS ICONです。ピュアなCSSだけで作られたアイコンセットです。

## CSS ICONの使い方

一例です。フラット、かつ細い線で書かれたフォントになります。

![](http://images.moongift.jp/2018/03/open-source-css%20icon-3_thumb.dca394b65ec327d40851492f2cdc2386.png "http://images.moongift.jp/2018/03/open-source-css%20icon-3.dca394b65ec327d40851492f2cdc2386.png")

使い方です。色などは自由に変更できます。

![](http://images.moongift.jp/2018/03/open-source-css%20icon-2_thumb.676c79854906f41613ec9a28169b46df.png "http://images.moongift.jp/2018/03/open-source-css%20icon-2.676c79854906f41613ec9a28169b46df.png")

白黒反転版もあります。

![](http://images.moongift.jp/2018/03/open-source-css%20icon-1_thumb.ce1c19aab8c3584b892523ec314b0aee.png "http://images.moongift.jp/2018/03/open-source-css%20icon-1.ce1c19aab8c3584b892523ec314b0aee.png")

CSS ICONはアイコンフォントと同様に線の色を変えられるのが利点と言えそうです。大きさも自由ですし、縦横のサイズを変えることだってできます。回転させたりすることもできますし、CSSならでは他の技術と組み合わせた使い方もできるでしょう。

CSS ICONはCSS製のオープンソース・ソフトウェア（Creative Commons）です。

[CSS ICON -- project by Wenting Zhang](http://cssicon.space/)
[wentin/cssicon: icon set made with pure css code, no dependencies, "grab and go" icons](https://github.com/wentin/cssicon)

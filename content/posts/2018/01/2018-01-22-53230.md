開発者にとってMarkdownは便利な記法ですが、それ以外の人たちにとってはルールを覚えなければならず、面倒に感じられてしまいます。かといってWYSIWYGなツールバーを設けると開発者が邪魔に感じてしまうでしょう。
そこで使ってみたいのがTOAST UI Editorです。MarkdownとWYSIWYGを切り替えられるエディタです。

## TOAST UI Editorの使い方

エディタです。上部にツールバーがついています。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-7_thumb.afbb56f6d20ba60778a7709df6814abf.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-7.afbb56f6d20ba60778a7709df6814abf.png")

日本語も問題なく入力できます。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-6_thumb.1e938bedee2e1a9899307500d821d371.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-6.1e938bedee2e1a9899307500d821d371.png")

テーブルをセル数をグラフィカルに決められるのは便利です。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-5_thumb.8022c08397fd57009b8c39b00cbab1c5.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-5.8022c08397fd57009b8c39b00cbab1c5.png")

Markdownの表示に切り替えることもできます。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-2_thumb.178a98dcac6e2ea2ff317c10eb124bde.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-2.178a98dcac6e2ea2ff317c10eb124bde.png")

プレビューの表示もできます。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-4_thumb.8adda0a30baa005085204df07d1e852d.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-4.8adda0a30baa005085204df07d1e852d.png")

反映はリアルタイムに行われます。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-3_thumb.f80975e213a5b9bdd578a378f609911d.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-3.f80975e213a5b9bdd578a378f609911d.png")

機能拡張によってグラフを追加したりできます。

![](http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-1_thumb.b23b39aa037f519c1a75e93fffafa1b4.png "http://images.moongift.jp/2018/01/open-source-toast%20ui%20editor-1.b23b39aa037f519c1a75e93fffafa1b4.png")

TOAST UI Editorは一般ユーザが使いやすいWYSIWYGと開発者が使いやすいMarkdown入力の両方をサポートしています。Markdownで使える機能しかツールバーで指定できませんが、大抵のコンテンツ作成には事足りるでしょう。

TOAST UI EditorはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

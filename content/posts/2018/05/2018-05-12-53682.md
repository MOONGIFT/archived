Markdownエディタは数多く存在します。多くは二画面形式になっており、左側がエディタ、右側がプレビューになっています。しかし、この場合は編集している内容とその結果の両方を逐次確認しないといけません。
Slate markdown editorはそういった手間なく、その場で表示に反映されるので初心者にも分かりやすい仕組みになっています。

## Slate markdown editorの使い方

Slate markdown editorの画面。左側がエディタ、右側には変換後のHTMLが表示されています。

![](http://images.moongift.jp/2018/05/open-source-slate%20markdown%20editor-3_thumb.0af95ba6a267f20f5e7e60c0ae7c7f74.png "http://images.moongift.jp/2018/05/open-source-slate%20markdown%20editor-3.0af95ba6a267f20f5e7e60c0ae7c7f74.png")

画像は選択すると削除できます。

![](http://images.moongift.jp/2018/05/open-source-slate%20markdown%20editor-2_thumb.334a7b0ef096d80be633fc52f9b3d475.png "http://images.moongift.jp/2018/05/open-source-slate%20markdown%20editor-2.334a7b0ef096d80be633fc52f9b3d475.png")

新しい画像を埋め込むこともできます。

![](http://images.moongift.jp/2018/05/open-source-slate%20markdown%20editor-1_thumb.bbd3e89339c576314027476970884434.png "http://images.moongift.jp/2018/05/open-source-slate%20markdown%20editor-1.bbd3e89339c576314027476970884434.png")

Slate markdown editorは多数のキーボードショートカットが用意されており、その多くはMarkdown記法を入力した後でスペースキーを入力すると反映されます。慣れると、Markdown記法を使いながらその場で表示がどうなるか確認できるのでさくさくと書けるようになるでしょう。WYSIWYGにありがちなツールバーで選択するのとはまた違った入力方式です。

Slate markdown editorはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

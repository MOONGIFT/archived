GitHubの活動計（Contribution Chart）は開発者としてのやる気を喚起してくれます。すべて埋めてやろうと躍起になる開発者もいるのではないでしょうか。しかし直近の情報しか見えないのが残念です。
そこで使ってみたいのがGitHub Contribution Chart Generatorです。これまでのすべての活動を可視化してくれます。

## GitHub Contribution Chart Generatorの使い方

ユーザ名を入れるだけで使えます。

![](http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-4_thumb.3bd428b5ab42de5e4ec6120ac36b8e4f.png "http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-4.3bd428b5ab42de5e4ec6120ac36b8e4f.png")

例えばこんな感じ。まばらだ…。

![](http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-3_thumb.b37bfbbfba822b8fbf21a893b158fd1f.png "http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-3.b37bfbbfba822b8fbf21a893b158fd1f.png")

テーマを変えられます。

![](http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-2_thumb.532f4f7a4ff13cb75fe91508891f3423.png "http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-2.532f4f7a4ff13cb75fe91508891f3423.png")

ブラック系。

![](http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-1_thumb.a7d0c77b6502d7b854a4aa8455215ce9.png "http://images.moongift.jp/2018/05/open-source-github%20contribution%20chart%20generator-1.a7d0c77b6502d7b854a4aa8455215ce9.png")

GitHub Contribution Chart Generatorを使えば、自分のこれまでの開発活動がどうなっているのか一目で分かります。上向いていれば良いですが、徐々に開発しなくなってしまっている…なんて方は開発者としては要注意ですね。

GitHub Contribution Chart GeneratorはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

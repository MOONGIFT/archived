GitHubがマイクロソフトに買収されました。かつてのマイクロソフトとは違うとはいえ、Skypeのことを思うと一抹の不安を拭えずにはいます（CEOも代わるので）。今後数年は色々あるかも知れません。
そんなGitHubをWindows XPにしてしまうデザインテーマがGithub XPです。

## Github XPの使い方

インストールした状態です。確かにWindows XP…右下にいるクリップが憎らしい。

![](http://images.moongift.jp/2018/06/open-source-github%20xp_thumb.257fd60762d94c9ca6fa318f6ea2e428.png "http://images.moongift.jp/2018/06/open-source-github%20xp.257fd60762d94c9ca6fa318f6ea2e428.png")

GitHub XPはジョークシェアで、特にインストールしたから何かある訳ではありません。クリップに質問ができる訳でもありません。タスクバーも通常のナビゲーションバーが下に移っただけです。自分のWebブラウザが突然これになっていたら、相当驚きそうです。

GitHub XPはCSS製のソフトウェア（ソースコードは公開されていますがライセンスは明記されていません）です。

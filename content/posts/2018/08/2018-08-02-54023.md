Google Chromeで新しいタブを開くと、最近見るWebサイトであったり、検索欄が表示されます。しかし、Web検索であればアドレスバーからやりますし、履歴もすぐにアクセスできます。この画面からあえてアクセスすることは少ないのではないでしょうか。
そこでもっと役立つ情報を表示してみましょう。それがMarkdown New Tabです。

## Markdown New Tabの使い方

Markdown New Tabをインストールすると新しいタブがMarkdownエディタになります。

![](http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-5_thumb.88ebe2e78bebca4de4f8545a0e0d14d1.png "http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-5.88ebe2e78bebca4de4f8545a0e0d14d1.png")

編集画面です。コマンド + Xで編集モードになります。

![](http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-4_thumb.be814b2f58a76aac26cb5cbc5ca7eb60.png "http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-4.be814b2f58a76aac26cb5cbc5ca7eb60.png")

日本語も使えます。

![](http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-3_thumb.69ffb89183e9f02953912572d9736dca.png "http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-3.69ffb89183e9f02953912572d9736dca.png")

設定です。リビジョン管理もされます。

![](http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-2_thumb.0370042c90c300a3f9f9d062ea3b71aa.png "http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-2.0370042c90c300a3f9f9d062ea3b71aa.png")

設定次第でこんな感じにもなります。

![](http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-1.f867ebe91835dd32a0b99122b2b5eed4.gif "http://images.moongift.jp/2018/07/open-source-markdown%20new%20tab-1.f867ebe91835dd32a0b99122b2b5eed4.gif")

Markdown New Tabを使って、メモはもちろんタスクを書いておくこともできます。URLを貼り付けるのも良いでしょう。気になったことがあれば、新しいタブを開いてメモして閉じてしまうだけです。

Markdown New TabはGoogle Chrome用のオープンソース・ソフトウェア（MIT License）です。

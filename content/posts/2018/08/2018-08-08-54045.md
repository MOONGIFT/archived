画像編集と言えばPhotoshopを思い浮かべる人が多いでしょう（またはGIMP）。しかしPhotoshopのすべての機能を満遍なく使いこなしている方はほとんどいないはずです。それどころか、無料のソフトウェアでも実は十分だったという方もいるでしょう。
そこで注目したいのがTOAST UI Image Editorです。Webベースながら十分な画像編集を行えるソフトウェアです。

## TOAST UI Image Editorの使い方

編集画面です。グレースケール、セピアなどの編集は選択するだけでできます。

![](http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-4_thumb.584cb442a9764906c42a52b4f3002536.png "http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-4.584cb442a9764906c42a52b4f3002536.png")

クロップです。

![](http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-3_thumb.221fc1985902065d39e1a2aa878cb3af.png "http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-3.221fc1985902065d39e1a2aa878cb3af.png")

さらにノイズを乗せたりできます。

![](http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-2_thumb.92b5749bdbf1962cf22f03834867d590.png "http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-2.92b5749bdbf1962cf22f03834867d590.png")

スタンプもあります。

![](http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-1_thumb.18d8dad401d4c665ce2f6a611af0b766.png "http://images.moongift.jp/2018/07/open-source-toast%20ui%20image%20editor-1.18d8dad401d4c665ce2f6a611af0b766.png")

TOAST UI Image Editorはまだ機能は多くありませんが、今後に期待したいソフトウェアです。HTML5とCanvasを組み合わせることで、こんな優れた画像編集が実現できます。非常に面白いソフトウェアです。

TOAST UI Image EditorはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

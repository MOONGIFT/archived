良いデザインは真似してみると、その良さやエッセンス、理由などが見えてくるものです。細部にまでこだわったデザインはやはり美しく、使いやすいUIやUXになるものでしょう。
今回はReact Site Navというライブラリを紹介します。Stripeのメニューを真似したUIコンポーネントです。

## React Site Navの使い方

実際に動かしているところです。メニューの子項目がアニメーションしながら表示されます。

![](http://images.moongift.jp/2018/08/open-source-react%20site%20nav-2.a7a89ac19bd58144600521a05c506123.gif "http://images.moongift.jp/2018/08/open-source-react%20site%20nav-2.a7a89ac19bd58144600521a05c506123.gif")

横に移動した時には、いちいちアニメーションしないのも良い感じです。

![](http://images.moongift.jp/2018/08/open-source-react%20site%20nav-1.dc50c52b547cf5808f3fea726a2df86d.gif "http://images.moongift.jp/2018/08/open-source-react%20site%20nav-1.dc50c52b547cf5808f3fea726a2df86d.gif")

React Site Navはメニューがフリップしながら表示されるので目を引きます。アニメーション自体は高速なので、都度表示されたとしても、それほどストレスにはならないでしょう。何より普段は隠れているので気にならないのが良いです。

React Site NavはJavaScript製のオープンソース・ソフトウェア（MIT License）です。

スマートフォンで文字を入力するのは面倒ですが、画面にタッチ操作できるので指で絵を描いたりするのは簡単です。写真のレタッチであったり、自分が書いた絵をそのままアップロードしたりと絵には様々な使い道があります。
今回紹介するのはHTML5 Canvas Drawing App、Webブラウザを使ったドローアプリです。

## HTML5 Canvas Drawing Appの使い方

メイン画面です。色の変更ができます。

![](http://images.moongift.jp/2018/07/open-source-html5%20canvas%20drawing%20app-2_thumb.7ed70917c5d14ba0aeaf905269953841.png "http://images.moongift.jp/2018/07/open-source-html5%20canvas%20drawing%20app-2.7ed70917c5d14ba0aeaf905269953841.png")

マウス操作の他、タッチ操作でも描けます。

![](http://images.moongift.jp/2018/07/open-source-html5%20canvas%20drawing%20app-1_thumb.99d91d01aec209c81092bc5eea496537.png "http://images.moongift.jp/2018/07/open-source-html5%20canvas%20drawing%20app-1.99d91d01aec209c81092bc5eea496537.png")

HTML5 Canvas Drawing AppはデスクトップのWebブラウザはもちろん、iOS/Android/Windows Phoneなどでも動作します。Webアプリに組み込んだり、そこからさらに発展してハイブリッドアプリで使ってみるのも面白そうです。

HTML5 Canvas Drawing AppはHTML5/JavaScript製のオープンソース・ソフトウェア（MIT License）です。

[krisrak/html5-canvas-drawing-app: Sketchpad app using html5 canvas to draw using touch or mouse, works on iOS, Android, Window Phone and browser](https://github.com/krisrak/html5-canvas-drawing-app)
